## 8. Ядро содержания

### 8.1. Ключевые понятия

| Понятие | Определение (для ребёнка) |
|---------|--------------------------|
| Логическая задача | Задача, где из условий (подсказок) можно вывести единственно верный ответ |
| Условие / подсказка | Факт в задаче, который сужает круг возможных ответов |
| Единственное решение | У задачи ровно один вариант ответа, который не противоречит ни одному условию |
| Лишнее условие | Подсказка, которую можно убрать, и задача всё равно решается однозначно |
| Противоречивое условие | Подсказки, которые несовместимы — задача не имеет решения |
| Недостаточные условия | Подсказок мало — задача имеет несколько решений |
| Тест задачи | Проверяем: решается ли задача? Однозначно ли? Чужим человеком? |

### 8.2. Три критерия хорошей задачи (плакат)

```
✅ ХОРОШАЯ ЗАДАЧА:
  1. РЕШАЕМА     — условия не противоречат друг другу
  2. ОДНОЗНАЧНА  — ровно одно решение, не больше
  3. НЕОБХОДИМА  — каждое условие нужно; убери одно — появятся варианты
```

### 8.3. Обязательные умения (к концу модуля)

1. Анализировать чужую задачу: найти лишние, противоречивые, недостающие условия
2. Строить структуру задачи: персонажи, объекты, пул условий
3. Проверять задачу методом решения: заполнить таблицу / диаграмму / таблицу проверки
4. Дорабатывать задачу по результатам теста
5. Представлять задачу классу: объяснять решение и обосновывать каждое условие

### 8.4. Типы задач, которые может выбрать группа

| Тип | Инструмент | Пресквизит |
|-----|-----------|-----------|
| Задача на перебор с таблицей (кто что?) | Таблица подстановки | НШ-01 |
| Задача на закономерность | Правило ряда | НШ-02 |
| Задача с диаграммой Эйлера | Два круга + распределение | НШ-03 |
| Задача «рыцари и лжецы» | Таблица предположений | НШ-04 |
| Стратегическая задача (найди выигрышный ход) | Таблица В/П | НШ-05 |
| Комбинированная задача (два типа) | По выбору группы | Любые 2 |

### 8.5. Алгоритм «5 шагов автора» (плакат)

| Шаг | Действие |
|-----|----------|
| 1 | **Придумай мир.** Кто участвует? Что нужно найти? Сколько объектов? |
| 2 | **Определи ответ.** Сначала придумай правильный ответ — потом условия к нему |
| 3 | **Добавь условия.** По одному. После каждого — проверяй: сколько решений осталось? |
| 4 | **Останови вовремя.** Когда осталось 1 решение — стоп. Лишние условия убери |
| 5 | **Протестируй.** Отдай задачу другой группе. Решили? Правильно? Нашли лазейки? |

## 10. Поурочные сценарии

---

### Занятие 1: «Что делает задачу хорошей?»

**Продолжительность:** 35–40 минут
**Цель:** понять три критерия хорошей задачи через анализ готовых примеров; выбрать тип задачи для проекта

#### Акт I: ЗАХВАТ (5 минут)

**0–2' | Сюжетный захват: Письмо из издательства**
Формат: фронтально | Техника: Артефакт

*«Нам пришёл запрос из издательства "Задачник Корифея". Они собирают сборник логических задач от юных авторов. Лучшие задачи будут опубликованы. Требование одно: задача должна быть корректной — решаемой, однозначной и без лишних условий.»* Вешает плакат «3 критерия». Раздаёт авторские дневники.

**2–5' | Первая победа: реши за 30 секунд**
Формат: фронтально | Техника: Быстрая задача

Простая задача на перебор: «Аня, Боря, Вася — кошка, собака, рыбка. Аня не любит собак. Вася — аллергик (не рыбки). Боря — кошка.» Кто первый? Дети решают, учитель разбирает. *«Сколько условий? 3. Все нужны? Сейчас выясним.»*

#### Акт II: РАЗВИТИЕ (25 минут)

**5–15' | Анализ задач: ищем дефекты**
Формат: пары → общее обсуждение | Техника: Экспертиза

Каждая пара получает две задачи — одну хорошую и одну с дефектом. Дефекты разные у разных пар:

- *Лишнее условие:* «Катя, Лёня, Маша — яблоко, банан, груша. Катя не банан. Лёня — груша. Маша не яблоко и не груша.» → условие «Маша не яблоко» лишнее, задача решается без него.

- *Противоречие:* «Дима, Егор — чай, кофе. Дима пьёт чай. Дима не пьёт чай.» → решений нет.

- *Недостаток условий:* «Ира, Женя — книга, мяч. Ира любит читать.» → два решения: Ира-книга или Ира-мяч (читать можно по-разному).

- *Хорошая задача:* стандартная задача на перебор 3×3 из банка НШ-01.

Пары докладывают: нашли дефект? Какой именно?

→ Совет: дефектные задачи вызывают живой спор. Не торопите — именно в споре формируется понимание критериев

**15–19' | Физкультминутка: «Условие или не условие?»**
Формат: движение, жетоны | Техника: Голосование телом

Учитель читает фразы. «Нужное условие» — прыгнуть, «лишнее» — присесть, «противоречие» — руки крест-накрест:

- «Катя не любит синий цвет» (в задаче про цвета мячей) → нужное ✓
- «Петя — мальчик» (в задаче, где и так написано «Петя — мальчик») → лишнее
- «Вася старше Лены, и Вася моложе Лены» → противоречие

**19–28' | Формируем группы и выбираем тип задачи**
Формат: группы 2–3 человека | Техника: Свободный выбор с консультацией

Учитель раздаёт меню типов задач (таблица из раздела 8.4). Каждая группа:
1. Выбирает тип задачи
2. Обсуждает тему (животные, спорт, еда, города, сказочные персонажи…)
3. Записывает в авторский дневник: «Наша задача будет о ______. Тип: ______. Количество персонажей: ___.»

→ Совет: мягко отговаривайте от слишком сложных комбинированных задач — группы переоценивают силы. Лучше простая и корректная, чем сложная и сломанная.

**28–30' | Защита выбора: каждая группа — 20 секунд**
Называют тему и тип. Класс может задать один вопрос.

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**30–35' | Первая печать.**
Запись в авторский дневник: «Хорошая задача — это _______.» Интрига: *«Следующий раз — придумываем правильный ответ и первые условия.»*

### Занятие 2: «Строим задачу — от ответа к условиям»

**Продолжительность:** 35–40 минут
**Цель:** освоить принцип «сначала ответ, потом условия»; написать первый вариант задачи

#### Акт I: ЗАХВАТ (5 минут)

**0–5' | Разминка: «Обратная задача»**
Формат: пары | Техника: Инверсия

На доске — готовая таблица подстановки с заполненными «+» (ответ). Никаких условий. Задача: придумать минимум условий, чтобы к этому ответу пришёл любой другой человек.

*«Видите? Автор всегда работает от ответа назад — к условиям. Именно так мы будем создавать задачи.»*

#### Акт II: РАЗВИТИЕ (30 минут)

**5–10' | Шаг 1–2 алгоритма: придумываем мир и фиксируем ответ**
Формат: группы | Техника: Работа в авторском дневнике

Каждая группа заполняет шаблон в дневнике:

```
МОЯ ЗАДАЧА

Персонажи: _________________ (3–4 имени)
Объекты:   _________________ (3–4 предмета/свойства)
Правильный ответ (таблица / схема / позиция):
  [заполнить]
```

Учитель обходит группы, помогает. Проверяет: ответ записан ДО условий? Если нет — возвращает к шагу 2.

→ Совет: самая частая ошибка — группа начинает придумывать условия, не зафиксировав ответ. Тогда ответ «плывёт» вслед за условиями.

**10–20' | Шаг 3 алгоритма: добавляем условия по одному**
Формат: группы | Техника: Пошаговое добавление с проверкой

Каждая группа добавляет условия по одному. После каждого — обязательная проверка: сколько решений осталось? Используют инструмент своего типа задачи.

Шаблон в дневнике:

```
Условие 1: «_________» → решений осталось: ___
Условие 2: «_________» → решений осталось: ___
Условие 3: «_________» → решений осталось: ___
СТОП — осталось 1 решение ✅
```

Учитель обходит. Частая ошибка: добавили лишнее — мягко спрашивает «А убери это условие — задача ещё решается?»

**20–25' | Физкультминутка: «Живая таблица»**
Формат: движение | Техника: Физическая визуализация

Одна группа выходит к доске и показывает свою структуру: ученики-персонажи встают в строки, ученики-объекты — в столбцы. Учитель зачитывает условие — «персонажи» и «объекты» меняются, таблица заполняется.

**25–33' | Первый черновик задачи**
Формат: группы | Техника: Письменный черновик

Каждая группа записывает первый вариант задачи полным текстом — как будто для читателя:

```
Задача: «[Название]»
[Имена] [глагол] [объекты].
Условие 1: ...
Условие 2: ...
Условие 3: ...
Вопрос: Кто [что] [глагол]?
```

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**33–38' | Вторая печать.**
Группы зачитывают черновик вслух. Класс молча слушает — без оценок. Интрига: *«На следующем занятии — отдадим задачи чужим тестировщикам.»*

### Занятие 3: «Тест и доработка»

**Продолжительность:** 35–40 минут
**Цель:** протестировать задачу в чужих руках, найти дефекты и исправить

#### Акт I: ЗАХВАТ (5 минут)

**0–5' | Провокация: «Автор всегда думает, что задача понятна»**
Формат: фронтально | Техника: Демонстрация

Учитель показывает задачу с «невидимой» двусмысленностью: «Аня старше Бори. Боря старше Вани. Кто самый старший?» → Аня. Кажется очевидным. Но! «А если Аня — 9 лет, Боря — 10, Ваня — 8? Тогда Боря старший!» Нет, подождите — «старше» значит возраст… или рост? *«Автор знал, что имел в виду — читатель нет. Тест — это единственный способ узнать, понятна ли задача.»*

#### Акт II: РАЗВИТИЕ (25 минут)

**5–10' | Роли на занятие**
Каждая группа назначает роли карточками:
- **Автор** — объясняет задачу тестировщикам, если те совсем застряли (только после сигнала учителя)
- **Критик** — наблюдает за тестированием, записывает вопросы и затруднения
- **Тестировщик** — член другой группы, решает задачу вслух

**10–22' | Перекрёстное тестирование**
Формат: межгрупповой обмен | Техника: Тест в чужих руках

Группы обмениваются черновиками. Тестировщики решают задачу вслух (думают вслух — критики записывают). Авторы молчат и наблюдают. Через 8 минут — стоп.

Каждый тестировщик заполняет бланк теста:

```
БЛАНК ТЕСТА
Задача группы: _______
Я решил задачу: ДА / НЕТ / ЧАСТИЧНО
Нашёл решений: ___ (одно / несколько / нет)
Непонятные места: _______________________
Лишние условия (на мой взгляд): __________
Противоречия: ___________________________
Общее впечатление: _______________________
```

→ Совет: тишина авторов во время теста — ключевое правило. Они видят задачу своими глазами и не могут «подсказывать» читателю свои намерения.

**22–27' | Обратная связь: тестировщик → авторы**
5 минут на обсуждение каждой группы. Тестировщик зачитывает бланк, авторы задают уточняющие вопросы (не защищаются!).

→ Совет: учите принимать критику: «Спасибо, это полезно» — не «Нет, ты просто не понял».

**27–33' | Доработка**
Группы вносят правки: убирают лишние условия, уточняют формулировки, добавляют недостающие условия. Всё фиксируется в дневнике:

```
ЧТО ИЗМЕНИЛИ:
Было: «_______»  →  Стало: «_______»
Почему: _______
```

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**33–38' | Третья печать.**
Каждая группа называет одно главное изменение, которое они внесли. *«Следующий раз — финальная проверка и оформление для сборника.»*

### Занятие 4: «Финальная редактура»

**Продолжительность:** 35–40 минут
**Цель:** довести задачу до финального вида, оформить для сборника, провести взаимооценку по критериям

#### Акт I: ЗАХВАТ (5 минут)

**0–5' | Разминка: три задачи-загадки про задачи**
Формат: пары | Техника: Блиц

Три вопроса на 1 минуту каждый:
- «В задаче 2 условия. После первого — 4 варианта. После второго — тоже 4. Хорошая ли задача?» → Нет — второе условие лишнее.
- «В задаче 3 условия. Одно из них противоречит двум другим. Сколько решений?» → Ноль.
- «В задаче 4 условия. После всех четырёх — 2 варианта. Что нужно сделать?» → Добавить ещё одно условие.

#### Акт II: РАЗВИТИЕ (25 минут)

**5–12' | Финальная самопроверка по чек-листу**
Формат: группы | Техника: Самоаудит

Каждая группа проходит по чек-листу автора (рабочий лист 4):

```
ЧЕКЛИСТ АВТОРА
☐ Задача решается (нет противоречий в условиях)
☐ У задачи ровно одно решение
☐ Мы сами решили задачу по условиям и получили задуманный ответ
☐ Каждое условие необходимо (убери — появятся другие решения)
☐ Формулировки понятны человеку, который не знает ответа
☐ Есть вопрос в конце («Кто что? / Кто есть кто?»)
☐ Есть название задачи
```

Каждый «нет» → немедленная правка.

**12–17' | Физкультминутка: «Редакторский стол»**
Формат: движение | Техника: Галерейный обход

Черновики задач вывешиваются на стены. Все ходят по классу с листочками. На каждой задаче можно оставить стикер: 🟢 — «всё ясно», 🟡 — «вопрос», 🔴 — «нашёл дефект». Потом авторы собирают стикеры с задачи своей группы.

**17–25' | Оформление для сборника**
Формат: группы | Техника: Финальная запись

Каждая группа оформляет задачу на чистовом бланке (рабочий лист 4B):

```
━━━━━━━━━━━━━━━━━━━━━━━━━
Задача «[Название]»
Авторы: ___________________
Тип задачи: ________________

[Текст задачи]

Вопрос: ___________________

[Место для ответа — на обороте]
```

**25–30' | Взаимооценка по критериям**

Ответы на чистовых бланках запечатываются — откроются на презентации. Интрига: *«Финальное занятие — каждая группа выходит и защищает свою задачу перед всем классом.»*

### Занятие 5: «Презентация сборника» (контрольная точка)

**Продолжительность:** 35–40 минут
**Цель:** публично представить задачу, объяснить каждое условие, ответить на вопросы — финальная проверка понимания

#### Акт I: ЗАХВАТ (3 минуты)

**0–3'** *«Сегодня — день авторов. Каждая группа выходит, зачитывает задачу, даёт классу 3 минуты на решение — потом объясняет ответ и обосновывает каждое условие. Начинаем.»*

#### Акт II: ПРЕЗЕНТАЦИИ (28 минут)

**Формат:** последовательные выступления групп (~6–7 мин на группу)

**Структура выступления каждой группы:**

**Часть 1 — Задача (1 мин):** зачитывают задачу чётко и медленно. Класс решает самостоятельно (3 мин тишины).

**Часть 2 — Решение (2 мин):** группа показывает решение на доске (таблица / диаграмма / таблица предположений). Объясняет каждый шаг.

**Часть 3 — Защита условий (1–2 мин):** учитель или класс задаёт вопросы:
- «Зачем нужно условие №2? Что будет без него?»
- «Можно ли было решить задачу без условия №1?»
- «Есть ли другое решение, которое мы не заметили?»

Группа отвечает — это и есть контрольная точка.

→ Совет: не оценивайте задачи публично «лучше / хуже». Оценивайте умение объяснять и защищать. Даже простая задача с отличным объяснением — отличный результат.

**Возможные вопросы учителя к группе (выбирать по ситуации):**

- «Почему вы выбрали именно этот тип задачи?»
- «Что изменилось между первым черновиком и финальной версией?»
- «Если убрать условие [X] — сколько решений станет?»
- «Кто в группе придумал самое важное условие?»
- «Встречались ли у вас тупики при составлении? Как выбрались?»

#### Акт III: КУЛЬМИНАЦИЯ (7 минут)

**28–35' | Торжественное вручение.**

Пятая печать — каждой группе. Диплом «Автор логических задач» — каждому участнику. Задачи вклеиваются в «Сборник класса» (физический или цифровой).

Итоговая рефлексия всего маршрута: *«Вы прошли путь от решения задач — к созданию задач. Что оказалось сложнее? Что поняли про логику, чего не знали раньше?»*

## 11. Контрольная точка: критерии

### 11.1. Чек-лист (оценивается при презентации)

| Критерий | Да | Частично | Нет |
|----------|-----|----------|-----|
| Задача решается (нет противоречий) | ☐ | ☐ | ☐ |
| У задачи ровно одно решение | ☐ | ☐ | ☐ |
| Каждое условие необходимо (группа может доказать) | ☐ | ☐ | ☐ |
| Группа правильно и чётко объяснила решение | ☐ | ☐ | ☐ |
| Группа ответила на вопрос об условии | ☐ | ☐ | ☐ |

### 11.2. Уровни достижения

| Уровень | Условие | Награда |
|---------|---------|---------|
| Базовый | Задача решаема + однозначна, объяснение дано | 5-я печать + диплом «Автор» |
| Продвинутый | Все три критерия + уверенная защита условий | 5-я печать + диплом + задача в сборник |
| Олимпиадный | Комбинированная задача (два типа) или задача с «красивой» ловушкой | Диплом + специальная отметка «Задача-жемчужина» |

### 11.3. Если группа не справилась с корректностью

Мини-консультация (5 мин до презентации): вместе проходим чек-лист, исправляем самый критичный дефект. Группа выходит с исправленной версией — и это честно говорит классу: «Мы нашли ошибку и исправили». Это тоже ценный опыт.

## 12. Банк материалов для анализа

### 12.1. Задачи с дефектами (для занятия 1)

**Д-01. Лишнее условие:**
«Коля, Поля, Толя — велосипед, самокат, ролики. Коля не ездит на роликах. Поля — велосипед. Толя не велосипед и не ролики.»
→ Условие «Толя не велосипед и не ролики» лишнее: из условий 1 и 2 уже следует, что Коля — самокат, Толя — ролики.

**Д-02. Противоречие:**
«Рита и Сима — кошка и собака. Рита — кошка. Сима — кошка.»
→ Решения нет — обе не могут быть кошкой.

**Д-03. Мало условий:**
«Вера, Гера — гитара, барабан. Вере нравится музыка.»
→ Два решения (любое из двух инструментов может быть у Веры).

**Д-04. Двусмысленность:**
«Миша выше Гриши. Гриша старше Миши. Кто больше?»
→ «Больше» — по росту или возрасту? Нет однозначного ответа.

### 12.2. Хорошие задачи (эталоны по типам)

**Э-01. Таблица подстановки (3×3):**
«Алиса, Борис, Вера занимаются: рисование, плавание, шахматы. Алиса не умеет рисовать и не играет в шахматы. Борис — не плавание.»
→ Алиса — плавание, Борис — рисование, Вера — шахматы. Все три условия нужны.

**Э-02. Диаграмма Эйлера:**
«В классе 20 учеников. 12 любят математику, 10 любят рисование, 5 любят и то и другое. Сколько не любят ни то ни другое?»
→ 20 − (12 + 10 − 5) = 3.

**Э-03. Рыцари и лжецы:**
«Дон говорит: "Эва — лжец." Эва говорит: "Дон — лжец." Дон пришёл из деревни рыцарей.»
→ Дон рыцарь → «Эва лжец» правда → Эва лжец → «Дон лжец» ложь ✅.

**Э-04. Стратегическая задача:**
«В куче 10 камней, берём 1 или 2. Ты ходишь первым. Гарантируешь победу? Сколько берёшь и почему?»
→ Берёшь 1 → оставляешь 9 (кратное 3 = П-позиция для соперника). Да, гарантируешь.

### 12.3. Шаблоны задач для слабых групп

Если группа не может придумать тему — предложить готовую структуру на выбор:

**Шаблон А:** «[Имя1], [Имя2], [Имя3] держат питомцев: [кошку, собаку, рыбку]. [Условие1]. [Условие2]. [Условие3].»

**Шаблон Б:** «[Имя1] говорит: "[Имя2] — рыцарь." [Имя2] говорит: "[Имя1] — лжец." [Дополнительный факт].»

Группа вставляет имена, объекты и придумывает условия — структура уже готова.

## 13. Частые проблемы и решения

| Проблема | Решение |
|----------|---------|
| Группа начала с условий, а не с ответа | «Стоп. Запишите ответ в таблицу прямо сейчас. Потом — условия» |
| Задача не имеет решения | «Прочитайте все условия вместе. Какие два противоречат?» Убрать одно |
| Несколько решений после всех условий | «Нужно ещё одно условие. Смотрите: какие два варианта остались? Придумайте факт, который разделит их» |
| Слишком много условий | «Убери последнее условие. Задача ещё решается? Тогда оно лишнее» |
| Группа не может договориться | Карточки-роли: у каждого своя функция. Автор предлагает, Критик проверяет, Тестировщик решает — решение принимает большинством |
| На презентации не могут объяснить условие | «Хорошо, покажите таблицу. Что изменится, если убрать это условие?» — если не знают ответа, условие скорее всего лишнее |
| Задача слишком простая | Поощрить за корректность, предложить добавить один твист: условие «от противного» или ложную подсказку-ловушку |

## 15. Финальный продукт маршрута «Классика»

По завершении Л-НШ-07 класс имеет:

- **«Сборник задач класса»** — физический или цифровой сборник из задач всех групп
- **Карта прогресса** — каждый ученик видит пройденные модули и полученные звания
- **Навыковый профиль** — что умеет решать, а что создавать

Эти материалы могут использоваться:
- как задачи для следующего года («решите задачи, которые придумали ваши предшественники»)
- как входная диагностика для Пула Б (5–6 класс)
- как портфолио для участия в олимпиадах по логике
