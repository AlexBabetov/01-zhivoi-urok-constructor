## 1. Философия

«Логика Корифей» — это не один курс, а библиотека самодостаточных модулей по 4–6 занятий. Каждый модуль — завершённый интенсив с собственным стандартом результата, контрольной точкой и полным комплектом материалов. Модули собираются в маршруты под разные задачи: факультатив на четверть, годовой курс, интенсив на каникулах, включение в существующие предметы.

### 1.1. Принципы

| Принцип | Суть |
|---------|------|
| Модульность | Каждый модуль самодостаточен. Можно взять один или собрать цепочку из восьми. Результат каждого модуля ценен сам по себе |
| Возрастная адекватность | Два пула с принципиально разной дидактикой: игровая для 3–4 класса (конкретно-образное мышление) и дискуссионная для 5–6 класса (переход к абстрактному) |
| Баланс 50/50 | Для 5–6 класса формальная логика и критическое мышление переплетаются внутри каждого модуля |
| Живой урок 360 | Трёхактная структура (Захват → Развитие → Кульминация), смена деятельности каждые 5–10 минут, ученик работает больше учителя |
| Единый модуль | Совместимость с системой «Единый модуль»: паспорт, ядро содержания, контрольная точка, параллельные треки (очно/онлайн) |
| Масштабируемость | Конструктор открыт: новые модули добавляются, не ломая систему |

### 4.3. Правила сочетания

Модули внутри одного пула сочетаются свободно с учётом пререквизитов. Модули из разных пулов не смешиваются — дидактика слишком разная. Если у модуля есть пререквизит, он обязателен. Модули без пререквизитов можно брать в любом порядке. Между модулями можно делать паузы — каждый модуль замкнут.

| Поле | Значение |
|------|----------|
| Код модуля | Л-НШ-01 |
| Название | «Мир загадок» |
| Возраст | 3–4 класс (9–10 лет) |
| Объём | 4 занятия по 35–40 минут |
| Пререквизиты | Нет (входной модуль) |
| Ключевой вопрос | «Как разгадать загадку, если нужно подумать по порядку?» |

| Сюжетная рамка | «Детективное агентство Логик» — дети становятся детективами и раскрывают дела |
| Геймификация | Карта детектива: за каждое раскрытое дело — наклейка-лупа. 4 лупы = бейдж «Супер-детектив» |

---

## 8. Ядро содержания

### 8.1. Ключевые понятия

| Понятие | Определение (для ребёнка) |
|---------|--------------------------|
| Логическая задача | Задача, в которой нужно рассуждать, а не считать |
| Перебор | Проверка всех возможных вариантов по порядку |
| Исключение | Убираем то, что точно не подходит |
| Таблица подстановки | Строки = персонажи, столбцы = варианты. «+» = да, «–» = нет |
| Условие (подсказка) | Факт из задачи, который сужает круг возможностей |

### 8.2. Обязательные умения (к концу модуля)

1. Читать условие задачи и выделять подсказки (условия)
2. Заполнять таблицу подстановки: «+» = да, «–» = нет
3. Исключать невозможные варианты по условиям
4. Делать вывод: если в строке остался один вариант — ставить «+»
5. Объяснять своё решение: «Я узнал, что..., потому что...»

### 8.3. Алгоритм решения (плакат «5 шагов детектива»)

| Шаг | Действие |
|-----|----------|
| 1 | Прочитай задачу. Кто участвует? Что нужно узнать? |
| 2 | Нарисуй таблицу. Слева — персонажи, сверху — варианты |
| 3 | Читай подсказки по одной. Ставь «–» там, где точно нет |
| 4 | Если в строке остался один вариант — ставь «+»! |
| 5 | Проверь: все ли подсказки совпадают с ответом? |

## 10. Поурочные сценарии

### Занятие 1: «Первое дело»

**Продолжительность:** 35–40 минут
**Цель:** познакомить с логическими задачами и таблицей подстановки

#### Акт I: ЗАХВАТ (5 минут)

**0–2' | Сюжетный захват: Письмо от детектива**
Формат: фронтально | Техника: Письмо от персонажа

Учитель: *«Ребята, сегодня утром мне пришло письмо от знаменитого детектива. Он пишет, что ему нужны помощники — юные детективы, которые умеют думать. Готовы попробовать?»* Достаёт конверт, читает вслух: *«Дорогие ребята! Я — детектив Логик. У меня накопились нераскрытые дела. Мне нужна команда умных помощников. Если вы пройдёте 4 испытания — получите бейдж Супер-детектива!»*

Ученики: слушают, реагируют эмоционально, загораются

→ Совет: подготовьте настоящий конверт с наклейкой-лупой / сургучной печатью

**2–5' | Первая победа: мини-загадка на 2 объекта**
Формат: фронтально | Техника: Первая победа

Учитель: *«Вот первое испытание — разминка! У Коли есть кошка. У Маши нет кошки. У кого кошка?»* (Хором: «У Коли!») *«Ещё одна: Один шарик красный, другой синий. У Вани не красный. Какой у Вани?»* («Синий!») *«Легко! Но сейчас будет сложнее...»*

Ученики: отвечают хором, переживают успех

#### Акт II: РАЗВИТИЕ (25 минут)

**5–12' | Дело №1: Задача на 2 объекта (совместное решение с таблицей)**
Формат: фронтально, у доски | Техника: Моделирование учителем

Задача: «Аня и Боря купили мороженое: шоколадное и клубничное. Аня не любит шоколад. Кто что купил?» Рисует таблицу 2×2 на доске. Вместе заполняют: Аня/шоколад = «–», Аня/клубника = «+», Боря/шоколад = «+». Вывешивает плакат «5 шагов детектива».

→ Совет: здесь вводим алгоритм. Каждый шаг проговариваем и показываем на плакате

**12–17' | Физкультминутка-логика: «Прыгни или присядь»**
Формат: движение | Техника: Голосование ногами

*«Если ТОЧНО так — прыгаем. Если не обязательно — приседаем.»* Примеры: «Все кошки полосатые» (приседаем). «Все собаки — животные» (прыгаем). «Все птицы летают» (приседаем — пингвин!). 8 утверждений.

→ Совет: это не разминка — это тренировка оценки утверждений!

**17–25' | Дело №2: Задача на 3 объекта (работа в парах)**
Формат: пары | Техника: Парная работа + конверты

Задача: «Катя, Лена и Миша занимаются спортом: плавание, футбол, гимнастика. Катя не играет в команде. Миша боится воды. Кто чем?» В конверте — пустая таблица 3×3.

→ Совет: типичная ошибка — ставят «+» сразу. Напоминайте: «Сначала минусы!»

**25–30' | Обсуждение решения у доски**
Одна пара объясняет, остальные сверяют.

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**30–35' | Первая лупа + рефлексия**
Раздаёт карты детектива и первые наклейки. Вопросы: «Что помогало думать? Где в жизни можно так рассуждать?»

### Занятие 2: «Дело усложняется»

**Продолжительность:** 35–40 минут
**Цель:** закрепить таблицу 3×3, ввести самостоятельное решение

#### Акт I: ЗАХВАТ (5 минут)

**0–5' | Разминка-блиц: 3 мини-задачи на скорость**
Формат: фронтально, маркерные доски | Техника: Блиц-опрос

Задачи на 2 объекта: (1) «Петя старше Даши. Кто младше?» (2) «У Вани не красный шарик. Какой?» (3) «Мурзик толще Барсика. Кто толще?»

#### Акт II: РАЗВИТИЕ (25 минут)

**5–13' | Дело №3: Задача на 3 объекта с новым типом условий**
Формат: фронтально → индивидуально | Техника: Постепенное освобождение

Задача: «Лев, зебра и обезьяна. Зовут Боб, Рики и Сэм. Боб не лазает по деревьям. Рики полосатый.» Первое условие вместе, дальше — сами.

**13–18' | Физкультминутка: «Найди подсказку»**
Формат: движение | Техника: Квест по классу

6 карточек на стенах (3 настоящих, 3 ложных). Дети ходят, определяют настоящие.

→ Совет: ложные подсказки должны быть забавными!

**18–28' | Дело №4: Задача на 3 объекта (полностью самостоятельно)**
Формат: индивидуально | Техника: Самостоятельная работа + дифференциация

Задача: «Артём, Борис, Вадим — математика, рисование, музыка. Артём не рисует и не играет. Вадим не считает.» Быстрым — бонус на 4 объекта.

→ Совет: кто затрудняется — «Какой шаг алгоритма ты делаешь сейчас?»

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**28–35' | Вторая лупа.** Интрига: «На следующем — задачи на 4 объекта и карусель станций!»

---

### Занятие 3: «Большое расследование»

**Продолжительность:** 35–40 минут
**Цель:** задачи на 4 объекта, групповая работа с дифференциацией

#### Акт I: ЗАХВАТ (5 минут)

**0–5' | Чёрный ящик**
Коробка с 4 карточками-именами и 4 карточками-предметами. «Четыре подозреваемых!»

#### Акт II: РАЗВИТИЕ (25 минут)

**5–15' | Дело №5: Задача на 4 объекта (совместное начало → пары)**
Задача: «Алиса, Борис, Вера, Гена — яблоко, банан, вишня, груша. Алиса не бананы и не вишню. Вера — вишня. Гена не яблоки и не груши.» Таблица 4×4. Первое условие вместе, дальше пары.

**15–20' | «Живая таблица»**
4 ученика с карточками-именами, на полу — листы с фруктами. Класс командует: «Вера, встань на вишню!» Решают задачу телом.

→ Совет: визуализация исключения через движение — запоминается лучше всего

**20–30' | Карусель станций**
Три станции:
- 🟢 Базовая (3 объекта): «Дима, Егор, Женя — гитара, барабан, флейта»
- 🟡 Продвинутая (4 объекта): «Аня, Белла, Вика, Галя — Москва, Казань, Сочи, Пермь»
- 🔴 Олимпиадная (4 объекта, сложные условия): «Иван, Кирилл, Лёша, Марк — хоккей, баскетбол, плавание, теннис»

Переход по сигналу колокольчика каждые 3–4 мин.

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**30–35' | Третья лупа.** «Последнее дело — без моей помощи!»

---

### Занятие 4: «Экзамен детектива» (контрольная точка)

**Продолжительность:** 35–40 минут
**Цель:** самостоятельное решение — проверка стандарта результата

#### Акт I: ЗАХВАТ (3 минуты)

**0–3'** «Два дела: на 3 и на 4 персонажа. Работаете самостоятельно. Алгоритм на плакате. 20 минут.»

#### Акт II: КОНТРОЛЬНАЯ РАБОТА (25 минут)

**3–13' | Задача А (3 объекта):** «Оля, Павел, Рита — гитара, пианино, скрипка. Оля не играет на струнных. Павел играет на гитаре.» + место для объяснения: «Я узнал, что _____, потому что _____.»

**13–23' | Задача Б (4 объекта):** «Саша, Таня, Ульяна, Филипп — зима, весна, лето, осень. Саша не любит холод. Таня — весна. Филипп не лето и не весна.» Быстрым — задача-звёздочка.

**23–28' | Самопроверка по эталону.** Ответы на доске.

#### Акт III: КУЛЬМИНАЦИЯ (7 минут)

**28–35' | Награждение.** Задача А = 4-я лупа. А + Б = бейдж «Супер-детектив». Рефлексия всего модуля.

## 11. Контрольная точка: критерии

### 11.1. Чек-лист

| Критерий | Да | Нет |
|----------|-----|-----|
| Задача А (3 объекта) решена верно | ☐ | ☐ |
| Таблица подстановки заполнена корректно (задача А) | ☐ | ☐ |
| Есть письменное объяснение хода решения | ☐ | ☐ |
| Задача Б (4 объекта) решена верно | ☐ | ☐ |
| Таблица подстановки заполнена корректно (задача Б) | ☐ | ☐ |

### 11.2. Уровни достижения

| Уровень | Условие | Награда |
|---------|---------|---------|
| Базовый | Задача А решена верно | 4-я лупа + бейдж «Детектив» |
| Продвинутый | Задачи А и Б решены верно | 4-я лупа + бейдж «Супер-детектив» |
| Олимпиадный | А + Б + задача-звёздочка | Бейдж «Супер-детектив» + приглашение на олимпиаду |

### 11.3. Если ученик не справился

Мини-консультация (5 мин): вместе решают одну задачу на 3 объекта. Затем альтернативная задача А'. Каждый ребёнок завершает модуль с ощущением успеха.

## 12. Банк заданий

### 12.1. Базовый уровень (2–3 объекта)

**Б-01. Мороженое.** Аня и Боря: шоколадное и клубничное. Аня не любит шоколад. → Аня — клубничное, Боря — шоколадное.

**Б-02. Зоопарк.** Лев, зебра, обезьяна. Боб, Рики, Сэм. Боб не лазает. Рики полосатый. → Рики — зебра, Боб — лев, Сэм — обезьяна.

**Б-03. Предметы.** Артём, Борис, Вадим — математика, рисование, музыка. Артём не рисует и не играет. Вадим не считает. → Артём — математика.

**Б-04. Животные.** Маша, Петя, Серёжа — кошка, собака, хомяк. Маша не любит больших. Петя — аллергик (хомяк — исключение). → Петя — хомяк, Маша — кошка, Серёжа — собака.

**Б-05. Спорт.** Катя, Лена, Миша — плавание, футбол, гимнастика. Катя не в команде. Миша боится воды. → Миша — не плавание, Катя — не футбол.

### 12.2. Продвинутый уровень (4 объекта)

**П-01. Фрукты.** Алиса, Борис, Вера, Гена — яблоко, банан, вишня, груша. Алиса не бананы/вишню. Вера — вишня. Гена не яблоки/груши. → Вера — вишня, Гена — банан.

**П-02. Времена года.** Саша, Таня, Ульяна, Филипп — зима, весна, лето, осень. Саша не холод. Таня — весна. Филипп не лето/весна. → Таня — весна, Саша — лето.

**П-03. Города.** Аня, Белла, Вика, Галя — Москва, Казань, Сочи, Пермь. Аня не Москва/Сочи. Белла — Казань. Галя не Пермь. → Белла — Казань, Аня — Пермь.

### 12.3. Олимпиадный уровень

**О-01. Спортсмены.** Иван, Кирилл, Лёша, Марк — хоккей, баскетбол, плавание, теннис. Иван и Кирилл не водный спорт. Марк — хоккей. Баскетболист — друг Лёши. → Марк — хоккей, Лёша — плавание.

**О-02. День рождения.** Аня, Боря, Витя, Галя — январь, март, июнь, сентябрь. Аня раньше всех. Витя позже Бори, раньше Гали. Боря — весной. → Аня — январь, Борис — март, Витя — июнь, Галя — сентябрь.

## 13. Частые проблемы и решения

| Проблема | Решение |
|----------|---------|
| Ставит «+» сразу | «Сначала минусы! Плюс — когда остался один вариант.» |
| Не понимает условие | Перечитайте вместе: «Кто участвует? Что ищем?» |
| Пара работает неравномерно | Роли: один читает, другой заполняет. Меняются |
| Решили слишком быстро | Задача следующего уровня или объясните другой группе |
| Отказывается работать | Роль «проверяющего» — сверяет решения других |
| Два решения | Обсуждение: «Какого условия не хватает? Придумайте!» |
| Не хватает времени | Сократите карусель до 2 станций |
