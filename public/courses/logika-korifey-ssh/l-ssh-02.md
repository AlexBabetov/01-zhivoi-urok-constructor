## 2. Ядро содержания

### 2.1. Ключевые понятия

| Понятие | Определение (для ученика) |
|---------|--------------------------|
| Понятие | Слово или выражение, которое обозначает целый класс похожих объектов |
| Объём понятия | Все объекты, которые подпадают под это понятие (их может быть много или мало) |
| Содержание понятия | Признаки, которые обязательно есть у каждого объекта понятия |
| Род | Большая группа, к которой принадлежит объект («стул» — это «мебель») |
| Видовое отличие | То, чем данный вид отличается от всех остальных в том же роду («стул» — мебель для сидения на одном человеке со спинкой) |
| Определение | Формула: «X — это [род], который [видовое отличие]» |
| Диаграмма Эйлера | Схема кругов, где больший круг = больший объём понятия, меньший — внутри |

### 2.2. Обязательные умения (к концу модуля)

1. Рисует диаграмму Эйлера для двух понятий — объясняет, какое шире
2. Называет обязательные и случайные признаки для заданного понятия
3. Строит определение по формуле «[род] + [видовое отличие]»
4. Находит ошибку «слишком широко» или «слишком узко» в чужом определении
5. Находит ошибку «определение через то же самое» (порочный круг)
6. Проверяет своё определение: не включает ли лишнее? не исключает ли своё?

### 2.3. Именной алгоритм — «3 шага лексикографа» (плакат)

| Шаг | Действие |
|-----|----------|
| 1 | **Найди род:** к какой большой группе относится X? (X — это [вид] [рода]) |
| 2 | **Найди видовое отличие:** чем X отличается от всех других представителей рода? |
| 3 | **Проверь:** входит ли в определение лишнее? исключает ли оно всё чужое? |

→ Определение готово: «X — это [род], который / которое [видовое отличие]»

### 2.4. Прогрессия сложности

| Занятие | Содержание | Сложность |
|---------|-----------|-----------|
| 1 | Объём понятия: большой класс / малый класс, диаграммы Эйлера | Входная |
| 2 | Содержание понятия: признаки обязательные и случайные | Базовая |
| 3 | Структура определения: род + видовое отличие, строим сами | Основная |
| 4 | Ошибки в определениях: слишком широко / узко / через то же самое | Продвинутая |
| 5 | Контрольная: дать определение 3 понятиям + найти ошибки в чужих | Итоговая |

## 4. Поурочные сценарии

---

### Занятие 1: «Инопланетный словарь — размеры понятий»

**Продолжительность:** 40–45 минут
**Цель:** ввести понятия объёма, научить строить диаграммы Эйлера

#### Акт I: ЗАХВАТ (7 минут)

**0–2' | Сюжетный захват: Сигнал от инопланетян**
Формат: фронтально | Техника: Реальный контекст (сыгранный)

Учитель: *«Вчера учёные получили сигнал с планеты Логос-4. Инопланетяне хотят знать о Земле всё — и просят прислать энциклопедию. Но не обычную — а такую, где каждое слово объяснено так точно, что даже существо, которое никогда не было на Земле, поймёт. Наше Бюро определений получило заказ. Вы — учёные-лексикографы. Начинаем.»*

Раздаются карточки «Учёный Бюро определений».

**2–7' | Первая победа: «Кого больше?»**
Формат: фронтально | Техника: Первая победа

На доске два слова: **«животное»** и **«кот»**.
Учитель: *«На планете Логос-4 хотят знать: кого больше — животных или котов? Объясните инопланетянам, почему.»*

Ученики легко отвечают. Учитель рисует два круга: большой («животное»), внутри маленький («кот»). *«Это и есть объём понятия. Сегодня научимся его изображать.»*

#### Акт II: РАЗВИТИЕ (28 минут)

**7–15' | Блок А: диаграммы Эйлера — строим вместе**
Формат: фронтально → пары | Техника: Моделирование учителем

Три пары слов — учитель разбирает первую, пары — вторую, все вместе — третью:
1. «Транспорт» и «велосипед» → маленький внутри большого
2. «Птица» и «животное» → птица внутри животного
3. «Кошка» и «собака» → два круга рядом (не пересекаются), оба внутри «домашнее животное»

Затем пары сами рисуют диаграмму для: «мебель» / «стул» / «шкаф».

→ Совет: для занятия 1 — только отношения «вложен» или «рядом». Пересечения (например, «спортсмен» и «студент») — в занятии 2.

**15–20' | Физкультминутка-логика: «Встань в круг»**
Формат: движение | Техника: Живая диаграмма

3 больших круга на полу (обруч или верёвка): «Живое существо» / «Растение» / «Дерево». Учитель называет объект — ученики встают в правильный круг. «Дуб» — все три. «Ромашка» — первые два. «Камень» — ни один. «Гриб?» — спорное (грибы — отдельное царство). Обсуждение: вот почему важны точные определения!

**20–33' | Блок Б: парная работа — «10 понятий в энциклопедию»**
Формат: пары | Техника: Парная работа с карточками

Рабочий лист 1. 6 пар понятий — нарисовать диаграммы Эйлера и написать 1–2 предложения: «Понятие А шире, чем Б, потому что...»

Пары понятий:
1. Фрукт / яблоко
2. Страна / Россия
3. Книга / роман
4. Спорт / футбол
5. Наука / физика
6. Профессия / учитель

Быстрым — придумать свою пару и объяснить соседней группе.

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**33–40' | Первое определение в энциклопедию + рефлексия**
Каждый пишет одно предложение для энциклопедии на карточку: «[Понятие] — это...» (пока неточно, без алгоритма). Карточки вклеиваются в «Энциклопедию класса». Интрига: *«Следующий раз узнаем, как сделать определение точным — чтобы инопланетянин не перепутал яблоко с грушей.»*

### Занятие 2: «Обязательные и случайные признаки»

**Продолжительность:** 40–45 минут
**Цель:** научить отделять обязательные признаки понятия от случайных

#### Акт I: ЗАХВАТ (7 минут)

**0–3' | Разминка: диаграммы на скорость**
Формат: маркерные доски | Техника: Блиц

3 пары слов — нарисовать диаграмму (30 секунд каждая). Проверяем вместе.

**3–7' | Первая победа: «Что обязательно у кошки?»**
Формат: фронтально | Техника: Мозговой штурм + провокация

На доске: «КОШКА». Учитель: *«Инопланетянин видит кошку первый раз. Какие признаки нужно знать ОБЯЗАТЕЛЬНО, чтобы не перепутать с другим животным? Перечислите всё.»*

Фиксируем на доске. Учитель задаёт провокационные вопросы: «Кошка рыжая — обязательный признак?» (нет, бывают белые) / «У кошки есть лапы — обязательный?» (да) / «Кошка мурлычет — обязательный?» (спорно). *«Вот вы уже умеете разделять обязательное и случайное!»*

#### Акт II: РАЗВИТИЕ (28 минут)

**7–17' | Блок А: таблица признаков — обязательный / случайный**
Формат: фронтально → пары | Техника: Моделирование учителем

Рабочий лист 2. Таблица: Понятие | Признак | Обязательный / Случайный | Почему.

Учитель разбирает «стул» вместе с классом:
- 4 ножки → случайный (есть стулья на 1 опоре)
- для сидения → **обязательный** (без этого уже не стул)
- деревянный → случайный (бывают пластиковые)
- со спинкой → **обязательный** (без спинки — табурет)
- красный → случайный

Затем пары разбирают «велосипед» самостоятельно. Обсуждение: где возникли разногласия?

**17–22' | Физкультминутка-логика: «Обязательный или нет?»**
Формат: движение | Техника: Голосование телом

Учитель называет признак и понятие. Обязательный — встать, случайный — сесть, спорно — стоять на одной ноге. Быстро, 8–10 примеров.

Примеры: «Школа — учителя есть» / «Школа — здание красного цвета» / «Книга — страницы есть» / «Книга — твёрдая обложка» / «Автобус — едет» / «Автобус — жёлтый».

**22–35' | Блок Б: пересекающиеся понятия (диаграммы с пересечением)**
Формат: пары → обсуждение | Техника: Диаграммы Эйлера сложный уровень

Новая ситуация: «студент» и «спортсмен» — оба можно быть одновременно. Пересекающийся круг.

Пары рисуют диаграммы для:
1. «Отличник» и «спортсмен»
2. «Школьник» и «читатель»
3. «Учёный» и «женщина»
4. «Врач» и «музыкант»

Вопрос: что значит пересечение? (есть объекты, которые принадлежат обоим понятиям)

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**35–42' | Второе определение в энциклопедию + рефлексия**
Ученики улучшают своё определение из занятия 1 — теперь добавляют обязательные признаки. Рефлексия: *«Чем обязательный признак отличается от случайного? Почему это важно для определения?»*

### Занятие 3: «Строим определения»

**Продолжительность:** 40–45 минут
**Цель:** освоить формулу определения «род + видовое отличие»

#### Акт I: ЗАХВАТ (7 минут)

**0–3' | Разминка: 4 понятия — обязательные признаки**
Формат: маркерные доски | Техника: Блиц-опрос

4 понятия, 30 секунд на каждое — назвать 2 обязательных признака.

**3–7' | Первая победа: «Угадай по определению»**
Формат: фронтально | Техника: Первая победа (игра)

Учитель читает определение, ученики угадывают понятие:
*«Это транспорт, который движется по рельсам и перевозит много пассажиров под землёй.»* → метро

*«Это мебель для сидения одного человека со спинкой.»* → стул

*«Это птица, которая не умеет летать и живёт только в Антарктике.»* → пингвин

*«Вы уже умеете понимать определения. Сегодня научимся их создавать.»*

#### Акт II: РАЗВИТИЕ (28 минут)

**7–18' | Блок А: формула «3 шагов лексикографа» — разбор**
Формат: фронтально → пары | Техника: Моделирование + постепенное освобождение

Плакат с алгоритмом. Учитель разбирает «термометр»:
- Шаг 1: Род — прибор (термометр — это прибор)
- Шаг 2: Видовое отличие — для измерения температуры
- Проверка: «Прибор для измерения температуры» — это только термометр? Да. → Определение готово.

Пары строят определение для «словаря» вместе с учителем, затем «библиотеки» самостоятельно.

Фиксируем шаблон на доске: *«[Объект] — это [род], [который / которое / предназначенный / используемый] + [видовое отличие].»*

**18–23' | Физкультминутка-логика: «Нет-да-нет»**
Формат: движение, парная | Техника: Игровое угадывание

Пары встают. Один придумывает понятие, второй угадывает, задавая вопросы только про род и видовое отличие: «Это живое существо?» / «Это используется человеком?» / «Это для хранения?» Три раунда. Цель — за 5 вопросов угадать.

**23–35' | Блок Б: «Дописать словарь инопланетянина» — групповая работа**
Формат: группы по 3–4 | Техника: Групповая работа

Каждая группа получает 5 понятий. Задача: написать определения по формуле «3 шагов». После — обмениваются листами с другой группой, проверяют по трём вопросам: Есть ли род? Есть ли видовое отличие? Не включает ли лишнее?

Понятия: ножницы, холодильник, словарь, учебник, карандаш, велосипед, зеркало, барабан.

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**35–42' | Третье определение в энциклопедию + рефлексия**
Каждый строит одно определение по формуле — вклеивает в энциклопедию. Рефлексия: *«В чём разница между "описать" и "определить"? Почему важно называть сначала род?»*

### Занятие 4: «Ошибки в определениях»

**Продолжительность:** 40–45 минут
**Цель:** научить находить три типа ошибок в определениях

#### Акт I: ЗАХВАТ (7 минут)

**0–3' | Разминка: три определения — правильные или нет?**
Формат: фронтально | Техника: Провокация

На доске три «определения» из прошлого занятия (специально с ошибками):
1. «Стул — это предмет для сидения» (слишком широко — подходит и кресло, и диван)
2. «Собака — это четырёхлапое домашнее животное, которое лает и кусается» (лишние признаки)
3. «Словарь — это книга, в которой словари объясняются» (определение через то же самое)

*«Что не так? Попробуйте объяснить инопланетянину по этим определениям. Запутается?»*

**3–7' | Первая победа: «Слишком широко» — очевидный пример**
Первая ошибка — найти вместе. Пара минут, легко справляются все.

#### Акт II: РАЗВИТИЕ (28 минут)

**7–17' | Блок А: три типа ошибок — разбор с примерами**
Формат: фронтально | Техника: Моделирование

**Ошибка 1: слишком широко** — определение подходит не только данному понятию.
*«Кошка — это животное, которое живёт дома.»* — Под это подходит и собака, и попугай. Значит, определение слишком широко.
Тест: подберём другой объект, который тоже подпадает.

**Ошибка 2: слишком узко** — определение исключает реальные примеры данного понятия.
*«Кошка — это рыжее животное с зелёными глазами.»* — Не все кошки рыжие. Значит, слишком узко.
Тест: найдём реальный объект, который не подходит под определение, хотя должен.

**Ошибка 3: порочный круг (через то же самое)**
*«Логика — это наука, которая изучает, как мыслить логически.»* — Объясняем «логику» через «логически» — это ничего не объясняет.
Тест: если убрать исходное слово — останется что-то содержательное?

**17–23' | Физкультминутка-логика: «Назови ошибку»**
Формат: движение, команды | Техника: Физическая сортировка

3 стола: «Слишком широко» / «Слишком узко» / «Через то же самое». Карточки с определениями разбросаны по классу. Каждый берёт одну карточку, определяет ошибку и кладёт на нужный стол. Спорные — обсуждаем вместе.

**23–35' | Блок Б: «Агентство по исправлению определений» — пары**
Формат: пары | Техника: Исправление + объяснение

Рабочий лист 4. 8 ошибочных определений. Для каждого: а) назвать тип ошибки; б) исправить определение.

Примеры:
1. «Учитель — это человек, который работает в школе.» (слишком широко — директор тоже)
2. «Птица — это летающее позвоночное животное.» (слишком узко — страус не летает)
3. «Понятие — это то, что мы понимаем.» (через то же самое)
4. «Велосипед — это транспорт.» (слишком широко)
5. «Роман — это очень длинная книга.» (слишком широко — бывают длинные повести)
6. «Математика — это наука о математических объектах.» (через то же самое)
7. «Учебник — это книга в твёрдой обложке для школьников.» (слишком узко — бывают мягкие)
8. «Зеркало — это предмет, в котором видно отражение.» (слишком широко — в воде тоже)

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**35–42' | Четвёртое определение + peer review**
Ученики пишут определение для нового понятия — сосед проверяет на три ошибки. Рефлексия: *«Какая ошибка кажется вам самой опасной? Почему?»*
Интрига: *«Финальное испытание — три понятия для инопланетян. Справимся?»*

---

### Занятие 5: «Экзамен лексикографа» (контрольная точка)

**Продолжительность:** 40–45 минут
**Цель:** самостоятельная работа — определения и разбор ошибок

#### Акт I: ЗАХВАТ (3 минуты)

**0–3'** Плакат «3 шага» на виду. Учитель: *«Последнее задание для Бюро определений — финальные три статьи энциклопедии. Инопланетяне ждут. Алгоритм перед вами.»* Торжественно, тихо.

#### Акт II: КОНТРОЛЬНАЯ РАБОТА (30 минут)

**3–18' | Задание А (базовый уровень): три определения**

Три понятия. Для каждого: а) указать род; б) указать видовое отличие; в) написать полное определение по формуле.

Понятия: **термометр** / **библиотека** / **карандаш**

**18–28' | Задание Б (продвинутый уровень): найти ошибки**

Три определения с ошибками. Для каждого: а) назвать тип ошибки; б) написать исправленный вариант.

1. «Учебник — это книга для учёбы.» (слишком широко — все книги можно использовать для учёбы)
2. «Кот — это домашнее животное с серой шерстью и зелёными глазами.» (слишком узко)
3. «Наука — это научная деятельность учёных.» (через то же самое)

**28–30' | Задача-звёздочка (олимпиадный)**

*«Придумай понятие, для которого сложно найти видовое отличие — где граница нечёткая. Объясни, почему это сложно, и всё равно попробуй дать определение.»*

(Примеры для учителя: «друг», «красота», «справедливость» — понятия с размытыми границами.)

#### Акт III: КУЛЬМИНАЦИЯ (10 минут)

**30–37' | Самопроверка + обсуждение сложных случаев**
Учитель показывает эталонные определения. Ученики проверяют. Разбираем 1–2 спорных.

**37–45' | Ачивка «Лексикограф» + рефлексия**
Выполнил задание А + Б — ачивка «Лексикограф». Рефлексия: *«Что умею теперь? Где в жизни пригодится умение давать точные определения?»*

Класс смотрит на собранную «Энциклопедию» — все определения вместе.

## 5. Контрольная точка: критерии

### 5.1. Чек-лист

| Критерий | Да | Нет |
|----------|----|-----|
| В задании А для каждого понятия указан корректный род | ☐ | ☐ |
| В задании А видовое отличие точно выделяет данное понятие среди прочих | ☐ | ☐ |
| В задании А определение сформулировано по формуле без ошибок | ☐ | ☐ |
| В задании Б все три типа ошибок названы верно | ☐ | ☐ |
| В задании Б исправленные определения корректны | ☐ | ☐ |

### 5.2. Уровни достижения

| Уровень | Условие | Награда |
|---------|---------|---------|
| Базовый | Задание А: все три определения по формуле | Ачивка «Лексикограф» |
| Продвинутый | А + Б: все ошибки найдены и исправлены | Ачивка «Лексикограф» + статус «Главный лексикограф» |
| Олимпиадный | А + Б + задача-звёздочка | Специальная отметка «Лексикограф-эксперт» |

### 5.3. Если ученик не справился

- Мини-консультация: вместе проходим шаг 1 и шаг 2 алгоритма на примере «кошки»
- Альтернативное задание А': одно понятие (простое — «ложка») + найти одну ошибку в определении
- **Правило:** каждый завершает модуль с ощущением успеха

## 6. Банк заданий

### 6.1. Базовый уровень

**Б-01.** Нарисуй диаграмму Эйлера: «животное» и «рыба». → Рыба внутри животного.

**Б-02.** Найди 3 обязательных признака понятия «школа». → Место обучения детей / с учителями / по программе.

**Б-03.** Дай определение «холодильнику» по формуле. → Бытовой прибор для хранения продуктов при низкой температуре.

**Б-04.** Дай определение «учебнику». → Книга, содержащая систематическое изложение знаний по учебному предмету.

**Б-05.** Дай определение «словарю». → Справочная книга, содержащая перечень слов с объяснением их значений или переводом.

**Б-06.** Нарисуй диаграммы: «транспорт», «воздушный транспорт», «самолёт». → Вложенные круги.

**Б-07.** Какой признак «велосипеда» обязательный: два колеса / педали / красный цвет / без мотора? → Два колеса + педали (обязательные).

**Б-08.** Дай определение «зеркалу». → Предмет с отполированной поверхностью, отражающей изображение.

### 6.2. Продвинутый уровень

**П-01.** «Компьютер — это прибор.» — какая ошибка? Исправь. → Слишком широко. Электронное устройство для обработки, хранения и передачи информации.

**П-02.** «Друг — это тот, кому доверяешь.» — ошибка? → Слишком широко (можно доверять незнакомцу). Объясни сложность и предложи лучшее определение.

**П-03.** Нарисуй диаграмму и дай определение «квадрату», «прямоугольнику», «четырёхугольнику». Покажи отношения объёмов.

**П-04.** «Математика — это наука о числах.» — ошибка? → Слишком узко (математика изучает не только числа). Исправь.

**П-05.** Придумай понятие, которое сложно определить, потому что его объём нечёткий. Попробуй дать определение. → Примеры: «молодость», «хорошая погода», «успех».

**П-06.** Дай определение «другу», «Родине», «справедливости». Объясни, почему эти определения труднее, чем «стул» или «термометр».

### 6.3. Олимпиадный уровень

**О-01.** «Живое существо — это то, что дышит.» Найди минимум 2 ошибки. Дай корректное определение. → Слишком узко (есть живые без кислородного дыхания) + слишком широко (огонь тоже «дышит» кислородом).

**О-02.** «Является ли "думающая машина" — компьютер — живым существом?» Построй аргумент через определения. Какие признаки «живого» она имеет / не имеет?

**О-03.** Придумай два понятия, диаграмма которых — пересекающиеся круги. Объясни, что находится в зоне пересечения и почему.

**О-04.** «Можно ли дать абсолютно точное определение любому понятию?» Приведи пример понятия с нечёткими границами и объясни, почему это философская проблема.

## 7. Частые проблемы и решения

| Проблема (что делает ученик) | Решение (что делает учитель) |
|-----------------------------|------------------------------|
| Называет случайный признак как обязательный | «Проверка: бывает ли [понятие] без этого признака? Если да — он не обязательный» |
| Пишет описание вместо определения | «Где твой РОД? Скажи: "это [род]..."» |
| Забывает видовое отличие — пишет только род | «Я знаю, что стул — мебель. Но чем стул отличается от дивана?» |
| Определение слишком длинное (5+ признаков) | «Оставь только те, без которых уже не [понятие]» |
| Не видит ошибку «через то же самое» | «Если убрать само слово из определения — объясняет ли оставшееся, что это такое?» |
| Путает объём и содержание | «Объём — сколько таких вещей. Содержание — что в каждой из них» |
| Не понимает, зачем энциклопедия инопланетянам | Спросите: «Если бы ты никогда не видел кошку — поможет ли тебе это определение?» |
| Быстро заканчивает | Дайте понятие с нечёткими границами: «молодость», «дружба», «успех» |

## 8. Соответствие методологии «Живой урок 360»

| Принцип | Реализация в модуле |
|---------|---------------------|
| Трёхактная структура | Захват → Развитие → Кульминация в каждом занятии; последнее занятие = контрольная точка |
| Смена деятельности 7–10 мин | 4–5 блоков по занятию: фронталь → физика → пары → группы |
| Ученик > учитель | 70%+ времени — самостоятельная, парная или групповая работа |
| Эмоции = память | Сюжет «Бюро определений» и энциклопедия для инопланетян — реальный артефакт модуля |
| Движение интегрировано | «Встань в круг», «Нет-да-нет», «Назови ошибку» — движение = задание |
| Дифференциация | 3 уровня задач (А / Б / звёздочка), банк заданий трёх уровней |
| 360° оценивание | Самопроверка (занятие 5), peer review (занятие 4), рефлексия в каждом занятии |
| Момент несогласия | Занятие 4: учитель специально показывает «ошибочные» определения, ученики опровергают |
| Реальный контекст | Знакомые предметы вокруг; понятие «энциклопедии» — реальная задача |
| Первая победа | Занятие 1: угадать «кого больше» — немедленный успех; занятие 3: угадать понятие по определению |
