# ЛОГИКА КОРИФЕЙ: Модуль Л-СШ-07 «Дебаты»

**5–6 классы (11–12 лет) • 5 занятий по 40–45 минут**

Методология «Живой урок 360» v5.1 • Школы Корифей • Март 2026

---

## 1. Паспорт модуля

| Поле | Значение |
|------|----------|
| Код модуля | Л-СШ-07 |
| Название | «Дебаты» |
| Возраст | 5–6 класс (11–12 лет) |
| Объём | 5 занятий по 40–45 минут |
| Пресквизиты | Л-СШ-03 (умение строить аргумент и работать с контраргументами) |
| Ключевой вопрос | «Можно ли победить в споре, не зная правды?» |
| Стандарт результата | Проводит аргументированное выступление на 2 мин с 3 аргументами; оценивает чужое выступление по рубрике |
| Сюжетная рамка | «Дебатный клуб» — ученики — члены клуба, где каждый должен уметь защищать любую позицию. Позицию получаешь жребием — не всегда ту, которую думаешь |
| Геймификация | Рейтинг дебатёра: за каждое занятие — очки по рубрике. Финальный рейтинг = ачивка «Дебатёр» / «Мастер дебатов» / «Гранд-мастер» |

## 2. Ядро содержания

### 2.1. Ключевые понятия

| Понятие | Определение (для ученика) |
|---------|--------------------------|
| Позиция | Чёткое утверждение — «за» или «против» по спорному вопросу |
| Тезис | Одно предложение — суть позиции, которую нужно защитить |
| Регламент | Правила дебатов: кто и сколько говорит, в каком порядке |
| Ответная реплика | Прямой ответ на аргумент оппонента — не новый аргумент, а разбор его |
| Сила аргумента | Насколько аргумент убедителен: факты, релевантность, логика |
| Рубрика оценивания | Таблица критериев, по которой оценивают качество выступления |

### 2.2. Обязательные умения (к концу модуля)

1. Формулирует чёткий тезис по любой заданной теме за 2 минуты
2. Строит 3 аргумента к тезису (из Л-СШ-03) за 5 минут
3. Произносит связное выступление по регламенту (2 минуты)
4. Слушает оппонента и отвечает на его конкретный аргумент, а не «мимо»
5. Завершает выступление резюме: тезис + самый сильный аргумент
6. Оценивает чужое выступление по рубрике — с обоснованием

### 2.3. Именной алгоритм — «5 шагов дебатёра» (плакат)

| Шаг | Действие |
|-----|----------|
| 1 | **Сформулируй позицию:** что именно ты хочешь доказать? (одно предложение) |
| 2 | **Подготовь 3 аргумента:** факты + логика, без эмоций |
| 3 | **Предугадай контраргументы:** что скажет оппонент? |
| 4 | **Слушай оппонента:** отвечай на его аргументы, не на воображаемые |
| 5 | **Завершай резюме:** повтори тезис + самый сильный аргумент |

### 2.4. Рубрика оценивания (используется с занятия 4)

| Критерий | 2 — отлично | 1 — частично | 0 — нет |
|----------|------------|--------------|---------|
| Тезис чёткий | Одно предложение, понятно | Есть, но расплывчатый | Не сформулирован |
| 3 аргумента | Все три, разные | 1–2 или повторяются | Аргументов нет |
| Факты, не эмоции | Опора на факты/логику | Смесь фактов и эмоций | Только эмоции |
| Ответ на оппонента | Отвечает на конкретный аргумент | Отвечает «мимо» | Игнорирует оппонента |
| Резюме | Тезис + лучший аргумент | Только тезис | Нет резюме |

**Максимум:** 10 очков. «Дебатёр» — 6+, «Мастер» — 8+, «Гранд-мастер» — 10.

### 2.5. Прогрессия сложности

| Занятие | Содержание | Сложность |
|---------|-----------|-----------|
| 1 | Что такое дебаты: правила, роли, регламент. Мини-дебаты по простой теме | Входная |
| 2 | Строим позицию: 3 аргумента к жребийной теме | Базовая |
| 3 | Слушаем и отвечаем: ответные реплики | Основная |
| 4 | Рубрика оценивания: учимся оценивать чужие выступления | Продвинутая |
| 5 | Контрольная: дебаты с peer review по рубрике | Итоговая |

## 3. Материалы модуля

| Материал | Описание |

| Плакат «5 шагов дебатёра» | Формат А3, висит в классе все 5 занятий |

| Рабочие листы 1–5 | По одному на занятие — бланки подготовки аргументов |
| «Банк тем» | Список 20 тем для дебатов — разной сложности |

### Банк тем (примеры)

**Простые (занятие 1–2):** «В школе надо носить форму» / «Домашние задания надо отменить» / «В школе должно быть больше перемен»

**Средние (занятие 3–4):** «Телефоны на уроках надо разрешить» / «Зоопарки надо закрыть» / «Оценки в школе мешают учёбе»

**Сложные (занятие 5):** «Социальные сети вредны для подростков» / «Обязательный предмет "Логика" нужен в каждой школе» / «Победить в споре важнее, чем найти правду»

## 4. Поурочные сценарии

---

### Занятие 1: «Добро пожаловать в Дебатный клуб»

**Продолжительность:** 40–45 минут
**Цель:** освоить правила и формат дебатов, провести первые мини-дебаты

#### Акт I: ЗАХВАТ (7 минут)

**0–2' | Сюжетный захват: Устав клуба**
Формат: фронтально | Техника: Реальный кейс

Учитель: *«Дебатный клуб — особое место. Здесь есть одно правило: ты защищаешь позицию, которую тебе дали жребием. Не ту, которую думаешь. Потому что настоящий дебатёр умеет убедительно защитить любую сторону. Это не обман — это мастерство. Сегодня начинаем.»*

Раздаются жетоны участника клуба.

**2–7' | Первая победа: «За» и «против» — быстро**
Формат: фронтально | Техника: Первая победа

Тема: «Лето лучше зимы». Учитель делит класс на две половины: одна — «за», другая — «против». 60 секунд — каждый придумывает один аргумент. Несколько называют. *«Вы уже умеете. Теперь добавим структуру и правила.»*

#### Акт II: РАЗВИТИЕ (28 минут)

**7–15' | Блок А: правила и формат дебатов**
Формат: фронтально | Техника: Объяснение + демонстрация

Формат мини-дебатов для класса (упрощённый):
1. **Жеребьёвка** — тема и сторона (за/против)
2. **Подготовка** — 3 минуты (5 шагов дебатёра, шаги 1–3)
3. **Выступление** — 2 минуты каждая сторона
4. **Ответные реплики** — 1 минута (шаг 4)
5. **Резюме** — 30 секунд (шаг 5)

Учитель показывает «плохой» пример выступления: эмоции вместо аргументов, уход от темы. Потом «хороший» — тезис + 2 аргумента + резюме.

**15–20' | Физкультминутка-логика: «Стань на сторону»**
Формат: движение | Техника: Физическое голосование

Учитель называет тему. Одна стена — «за», другая — «против», середина — «не знаю». Ученики встают. Потом учитель говорит: *«Теперь перейдите на ДРУГУЮ сторону и придумайте один аргумент ЗА эту сторону.»*

3 темы, каждый раз — переход и быстрый аргумент вслух.

**20–33' | Блок Б: первые мини-дебаты — тройки**
Формат: тройки | Техника: Мини-дебаты с наблюдателем

Тема: «В школе надо носить форму». Тройка: Дебатёр А (за) + Дебатёр Б (против) + Наблюдатель.

Подготовка: 3 минуты (5 шагов, шаги 1–2).
Выступление А: 1,5 минуты.
Выступление Б: 1,5 минуты.
Наблюдатель: называет один сильный аргумент каждого.

→ Совет: следите за временем строго — таймер помогает почувствовать регламент

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**33–40' | Первые очки + рефлексия**
Каждый наблюдатель называет, кто был убедительнее — и ПОЧЕМУ. Рефлексия: *«Легко ли защищать позицию, с которой не согласен? Что это развивает?»*
Интрига: *«Следующий раз — жеребьёвка и полноценная подготовка.»*

### Занятие 2: «Жеребьёвка — защищай любую сторону»

**Продолжительность:** 40–45 минут
**Цель:** построить 3 аргумента к незнакомой теме за ограниченное время

#### Акт I: ЗАХВАТ (7 минут)

**0–3' | Разминка: 1 аргумент за 30 секунд — 4 темы**
Формат: фронтально | Техника: Блиц

Учитель называет тему и сторону — каждый за 30 секунд пишет 1 аргумент на листке. 4 темы, потом 2–3 ученика зачитывают.

**3–7' | Первая победа: тянем жребий**
Конверты с темами. Учитель: *«Нет "лёгкой" и "сложной" стороны. Есть мастерство.»* Несколько учеников тянут и озвучивают, что вытащили — реакция класса.

#### Акт II: РАЗВИТИЕ (28 минут)

**7–18' | Блок А: стратегия подготовки — как искать аргументы**
Формат: фронтально | Техника: Моделирование учителем вслух

Учитель «думает вслух» по шагам 1–3, строя аргументы к теме, с которой лично не согласен.

Тема: «Зоопарки надо закрыть» (сторона: ПРОТИВ закрытия).
— Тезис: «Зоопарки необходимо сохранить.»
— Аргумент 1 (наука): «Зоопарки участвуют в программах сохранения видов под угрозой вымирания.»
— Аргумент 2 (образование): «Зоопарки — единственный способ для городских детей увидеть живых животных.»
— Аргумент 3 (практика): «Многие зоопарки уже перешли на условия, близкие к естественным.»
— Контраргументы: «Животные в клетках страдают» → ответ: «Современные зоопарки — не клетки.»

**18–23' | Физкультминутка-логика: «Банк аргументов»**
Формат: движение, класс | Техника: Совместная генерация

Одна тема на всех — класс встаёт. Каждый, кто придумал аргумент, делает шаг вперёд и называет. Нельзя повторять уже названный. Цель — набрать 10 разных аргументов (за любую сторону).

**23–35' | Блок Б: подготовка к дебатам — индивидуально**
Формат: индивидуально | Техника: Структурированная подготовка

Рабочий лист 2. Ученики тянут жребий — тема + сторона. 10 минут подготовки:
- Тезис (1 предложение)
- Аргумент 1 + факт/обоснование
- Аргумент 2 + факт/обоснование
- Аргумент 3 + факт/обоснование
- Возможные контраргументы оппонента

Затем — пары (противоположные стороны) проводят мини-дебаты (1,5 + 1,5 мин).

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**35–42' | Очки + рефлексия**
Пары называют: чей аргумент был самым сильным и почему? Рефлексия: *«Изменила ли подготовка ваше мнение по теме?»*

### Занятие 3: «Ответная реплика — слушаем и отвечаем»

**Продолжительность:** 40–45 минут
**Цель:** научить отвечать точно на аргументы оппонента, не уходить «мимо»

#### Акт I: ЗАХВАТ (7 минут)

**0–3' | Разминка: найди контраргумент за 1 минуту**
Учитель называет аргумент, ученики пишут контраргумент. 3 аргумента.

**3–7' | Первая победа: «Мимо или в точку?»**
Два ответа на один аргумент: один «мимо» (не отвечает на суть), один «в точку».
*«Оппонент сказал: "Домашние задания помогают закрепить материал." Ответ А: "Зато дети устают." Ответ Б: "Исследования показывают, что повторение на следующий день эффективнее, чем в тот же вечер."»*

*«Ответ А — мимо. Ответ Б — прямо. Научимся делать так.»*

#### Акт II: РАЗВИТИЕ (28 минут)

**7–17' | Блок А: техника ответной реплики**
Формат: фронтально | Техника: Моделирование + формула

Формула ответной реплики:
1. «Мой оппонент сказал, что [точная передача аргумента].»
2. «Однако это неверно / неполно / не доказывает тезис, потому что [контраргумент].»
3. «Следовательно, [вывод в пользу своей позиции].»

Разбираем три примера вместе — учитель произносит аргумент, ученики хором строят ответ по формуле.

→ Совет: шаг 1 (точная передача) — самый важный. Нельзя отвечать на искажённый аргумент — это соломенное чучело (связь с Л-СШ-04).

**17–23' | Физкультминутка-логика: «Пинг-понг аргументов»**
Формат: движение, пары стоя | Техника: Быстрый обмен

Пары встают напротив. Один говорит аргумент (15 сек). Второй — ответ по формуле (30 сек). Смена ролей. 3 раунда — разные темы. Темп быстрый, как настольный теннис.

**23–35' | Блок Б: полные дебаты с репликами — четвёрки**
Формат: четвёрки | Техника: Дебаты с полной структурой

Четвёрка: А1 (за) + А2 (за) + Б1 (против) + Б2 (против).
- Выступление А1: 2 минуты
- Выступление Б1: 2 минуты
- Ответная реплика Б2 на А1: 1 минута
- Ответная реплика А2 на Б1: 1 минута
- Резюме А1: 30 секунд
- Резюме Б1: 30 секунд

Наблюдатели — остальные ученики — отмечают: «Был ли ответ в точку или мимо?»

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**35–42' | Очки + разбор лучшей реплики**
Выбираем лучшую ответную реплику — объясняем почему. Рефлексия: *«Почему важно слушать оппонента, а не готовить следующий аргумент, пока он говорит?»*

### Занятие 4: «Рубрика — учимся оценивать»

**Продолжительность:** 40–45 минут
**Цель:** освоить рубрику оценивания, научиться давать и принимать обратную связь

#### Акт I: ЗАХВАТ (7 минут)

**0–3' | Разминка: по формуле — 3 ответные реплики**
Быстро: учитель называет аргумент, ученики отвечают вслух по формуле.

**3–7' | Первая победа: «Оцени это выступление»**
Учитель произносит короткое выступление (1 минута) — заведомо слабое: нет тезиса, один аргумент, только эмоции. Ученики инстинктивно чувствуют — слабо. *«Но как объяснить? Для этого нужна рубрика.»*

#### Акт II: РАЗВИТИЕ (28 минут)

**7–17' | Блок А: рубрика — разбор каждого критерия**
Формат: фронтально | Техника: Моделирование оценки

Раздаётся рубрика (5 критериев × 3 уровня = 10 очков max).

Учитель произносит два выступления — «слабое» и «сильное». Ученики заполняют рубрику для каждого. Сравниваем: 3 очка vs 9 очков. Обсуждаем расхождения — почему кто-то поставил иначе?

→ Совет: расхождения в оценках — это нормально и ценно. Обсуждение «почему ты поставил 1 вместо 2» = лучшее понимание критерия.

**17–23' | Физкультминутка-логика: «Судья встаёт»**
Формат: движение | Техника: Шкала голосованием

Учитель зачитывает отдельную фразу или реплику. Ученики показывают пальцами (0, 1 или 2) и, по сигналу, встают те, кто поставил 2. Объясняют — коротко.

4–5 фраз по разным критериям.

**23–35' | Блок Б: дебаты с live-оцениванием — группы**
Формат: группы по 5 | Техника: Дебаты + рубрика

В каждой группе: 2 дебатёра + 3 наблюдателя-судьи. Жребий — тема и сторона.

Дебатёры готовятся 4 минуты. Выступление (2+2 мин) + реплики (1+1 мин) + резюме (30+30 с).

Судьи во время выступления заполняют рубрику. После — озвучивают оценку + 1 конкретное улучшение.

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**35–42' | Очки + рефлексия**
Рефлексия: *«Тяжело ли оценивать другого? Какой критерий самый важный для вас лично?»*
Интрига: *«Финальные дебаты — всё по-настоящему.»*

### Занятие 5: «Финальный раунд» (контрольная точка)

**Продолжительность:** 40–45 минут
**Цель:** дебаты с оцениванием по рубрике — итог модуля

#### Акт I: ЗАХВАТ (3 минуты)

**0–3'** Плакат «5 шагов» на виду. Рубрики розданы. Жеребьёвка: каждый тянет тему + сторону. *«Финальный раунд. Защищайте позицию честно — даже если она не ваша.»*

#### Акт II: ДЕБАТЫ (30 минут)

**3–7' | Подготовка** (4 минуты — строго по таймеру)
Рабочий лист: тезис + 3 аргумента + контраргументы оппонента.

**7–35' | Дебаты в четвёрках**

Формат: пары противоположных сторон. 3 параллельных дебаты.

- Выступление А: 2 минуты
- Выступление Б: 2 минуты
- Ответная реплика А: 1 минута
- Ответная реплика Б: 1 минута
- Резюме А: 30 секунд
- Резюме Б: 30 секунд

Два наблюдателя (из другой четвёрки) заполняют рубрику для обоих.

**Задача-звёздочка (в ходе или после):**

*«Напиши 3 аргумента для противоположной стороны — той, которую ты НЕ защищал. Насколько они сильнее/слабее твоих?»*

#### Акт III: КУЛЬМИНАЦИЯ (10 минут)

**35–40' | Подсчёт очков и ачивки**

Судьи озвучивают оценки. Итоговый счёт:
- 6–7 очков → ачивка «Дебатёр»
- 8–9 очков → ачивка «Мастер дебатов»
- 10 очков → ачивка «Гранд-мастер»

**40–45' | Рефлексия модуля**
*«Что изменилось в том, как вы спорите? Можно ли победить в споре, не зная правды?»* (Возвращаемся к ключевому вопросу модуля.)

## 5. Контрольная точка: критерии

### 5.1. Чек-лист (для учителя + для самооценки)

| Критерий | Да | Нет |
|----------|----|-----|
| Тезис сформулирован одним чётким предложением | ☐ | ☐ |
| Приведено 3 разных аргумента | ☐ | ☐ |
| Аргументы опираются на факты/логику, а не только на эмоции | ☐ | ☐ |
| Ответная реплика отвечает на конкретный аргумент оппонента | ☐ | ☐ |
| Выступление завершается резюме (тезис + лучший аргумент) | ☐ | ☐ |

### 5.2. Уровни достижения

| Уровень | Условие | Награда |
|---------|---------|---------|
| Базовый | 6–7 очков по рубрике | Ачивка «Дебатёр» |
| Продвинутый | 8–9 очков | Ачивка «Мастер дебатов» |
| Олимпиадный | 10 очков + задача-звёздочка | Ачивка «Гранд-мастер» |

### 5.3. Если ученик не справился

- Мини-дебаты один на один с учителем — учитель задаёт лёгкую тему и помогает построить тезис + 2 аргумента
- Альтернатива: письменное выступление (тезис + 2 аргумента + резюме) вместо устного
- **Правило:** каждый завершает модуль с ощущением успеха

## 6. Банк заданий и тем

### 6.1. Темы по сложности

**Простые:** Форма в школе / Домашние задания / Длина каникул / Уроки физкультуры / Смартфоны на уроках

**Средние:** Зоопарки / Оценки в школе / Онлайн-обучение vs очное / Вегетарианство / Социальные сети

**Сложные:** Анонимность в интернете / Обязательное голосование / Ограничение рекламы / Победить в споре vs найти правду / «Логику» в школьную программу

### 6.2. Продвинутые задания

**П-01.** Напиши аргументы для обеих сторон по одной теме. Какая сторона оказалась сильнее? Почему?

**П-02.** Проанализируй выступление (видео публичной речи): найди тезис, аргументы, ответные реплики. Оцени по рубрике.

**П-03.** Тема, которую сложно разделить на «за» и «против» — почему некоторые вопросы не поддаются дебатам?

**П-04.** Построй «Идеальное выступление» — все 5 критериев рубрики на 2 балла. Запиши его текстом.

### 6.3. Олимпиадный уровень

**О-01.** «Побеждает сильнейший аргумент или сильнейший оратор?» Опиши ситуацию, где хорошая риторика побеждает слабый, но верный аргумент. Как это связано с ловушками мышления (Л-СШ-04)?

**О-02.** «Можно ли дебатировать об истине?» Какие темы поддаются дебатам (ценностные вопросы), а какие — нет (фактические)? Приведи примеры обоих.

**О-03.** Разработай свою рубрику оценивания дебатов — с другими или дополнительными критериями. Обоснуй каждый критерий.

## 7. Частые проблемы и решения

| Проблема (что делает ученик) | Решение (что делает учитель) |
|-----------------------------|------------------------------|
| Отказывается защищать «не свою» сторону | «Адвокат защищает клиента, не потому что согласен. Дебатёр — то же самое. Это тренировка мышления, не предательство убеждений» |
| Аргументы — только эмоции | «На каждый аргумент спрашивай: "Это факт или чувство? Можно ли проверить?"» (связь с Л-СШ-01) |
| Отвечает «мимо» — на воображаемый аргумент | «Повтори сначала: "Мой оппонент сказал..." — точно ли ты передал его слова?» |
| Теряется, забывает аргументы во время выступления | «Листок с тезисом и тремя аргументами — перед глазами всегда. Это не шпаргалка — это план» |
| Перебивает оппонента | «Правило клуба: ты победишь его словами — не громкостью. Пусть закончит» |
| Не знает, как начать резюме | Шаблон: «Таким образом, я настаиваю: [тезис]. Главный аргумент — [лучший аргумент].» |
| Слишком застенчив для публичного выступления | Начать с письменного варианта + выступление в тройке (маленькая аудитория) |
| Быстро заканчивает | «Найди в выступлении оппонента одну слабую посылку — и постройй развёрнутый ответ» |
