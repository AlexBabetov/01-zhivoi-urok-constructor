## 2. Ядро содержания

### 2.1. Ключевые понятия

| Понятие | Определение (для ученика) | Пример |
|---------|--------------------------|--------|
| **Суждение** | Утверждение, которое можно оценить как истинное или ложное | «Земля круглая» — суждение; «Закрой дверь!» — не суждение |
| **Умозаключение** | Получение нового суждения из двух или более уже известных | «Все рыбы дышат жабрами. Форель — рыба. → Форель дышит жабрами» |
| **Посылка** | Исходное суждение, из которого делается вывод | В примере выше — первые два суждения |
| **Вывод** | Новое суждение, которое следует из посылок | В примере — третье суждение |
| **Правильное умозаключение** | Вывод следует из посылок по верной логической форме | Структура «Все A есть B; X — A; → X — B» |
| **Парадокс** | Рассуждение, которое приводит к противоречию — хотя каждый шаг кажется верным | «Это суждение ложно» — истинно оно или ложно? |
| **Софизм** | Намеренно ошибочное рассуждение, созданное для обмана | «Все великие художники бедны. Я беден. Значит, я великий художник» |
| **Логическая связка** | Слово, связывающее суждения: И, ИЛИ, НЕ, ЕСЛИ... ТО | «Я приду, ЕСЛИ будет тепло» |

### 2.2. Обязательные умения (к концу модуля)

1. Отличает суждение от не-суждения (вопрос, приказ, восклицание)
2. Записывает простое умозаключение в форме «посылка 1 + посылка 2 → вывод»
3. Определяет, следует ли вывод из посылок (правильность формы)
4. Называет и объясняет не менее трёх классических парадоксов
5. Распознаёт не менее пяти типов бытовых и медийных софизмов
6. Применяет «5 шагов охотника» к незнакомому тексту

### 2.3. Именной алгоритм — «5 шагов охотника» (плакат)

```
┌─────────────────────────────────────────────────────────┐
│              5 ШАГОВ ОХОТНИКА ЗА ПАРАДОКСАМИ            │
├────┬────────────────────────────────────────────────────┤
│ 1  │ НАЙДИ СУЖДЕНИЯ — что именно утверждается?           │
│ 2  │ НАЙДИ УМОЗАКЛЮЧЕНИЕ — какой вывод делается?         │
│ 3  │ ПРОВЕРЬ ФОРМУ — структура «все A = B; X — A; → ?»  │
│ 4  │ ПРОВЕРЬ ПОСЫЛКИ — каждое исходное утверждение верно? │
│ 5  │ НАЗОВИ ЛОВУШКУ — как называется этот приём?         │
└────┴────────────────────────────────────────────────────┘
```

> **Правило охотника:** Рассуждение может быть неверным по форме (шаг 3) ИЛИ по содержанию (шаг 4). Нужно проверить оба.

## 4. Поурочные сценарии

---

### Занятие 1: «Как мы делаем выводы — и когда ошибаемся»

**Продолжительность:** 40–45 минут
**Цель:** ввести понятия суждения и умозаключения; показать разницу между правильным и неправильным умозаключением

---

#### Акт I: ЗАХВАТ (7 минут)

**0–2' | Сюжетный захват: письмо из Клуба**
Учитель зачитывает «письмо»:

> *«Добро пожаловать в Клуб охотников за парадоксами. Вы приняты на испытательный срок. Ваша первая задача — научиться видеть то, что большинство людей не замечают. Начнём с простого: каждый день вы делаете сотни умозаключений. Большинство — верные. Некоторые — ловушки. Различить их — ваше главное оружие.»*

**2–7' | Первая победа: два умозаключения**

На доске — два рассуждения. Ученики голосуют: верное или нет?

**Рассуждение А:**
> «Все собаки — животные.
> Рекс — собака.
> → Рекс — животное.»

Ученики: верное! ✓

**Рассуждение Б:**
> «Все собаки — животные.
> Кот — животное.
> → Кот — собака.»

Ученики: неверное! ✗ А почему? Ведь обе посылки — правда...

*Именно этим и занимается Клуб. Посылки правдивые. Форма — сломанная.*

---

#### Акт II: РАЗВИТИЕ (28 минут)

**7–16' | Блок А: суждение, посылка, вывод**
Формат: фронтально | Техника: строим вместе

Три компонента умозаключения — разбираем на доске:

```
ПОСЫЛКА 1:  Все металлы проводят электричество.
ПОСЫЛКА 2:  Медь — металл.
ВЫВОД:      → Медь проводит электричество.
```

Важные различия:
- Суждение ≠ вопрос, приказ, пожелание — суждение можно назвать «истинным» или «ложным»
- Умозаключение = получение НОВОГО знания из уже известного
- Вывод следует из посылок — или только *кажется*, что следует

**Правило формы:** форма умозаключения может быть верной, даже если суждения ложные — и наоборот:

| Посылки | Форма | Вывод верный? |
|---------|-------|---------------|
| Истинные | Правильная | Да |
| Истинные | Неправильная | Нет |
| Ложные | Правильная | Вывод формально следует — но он ложный |

Пример с ложными посылками и правильной формой:
> «Все кошки живут на Марсе. Мурзик — кошка. → Мурзик живёт на Марсе.»
*Форма правильная, но посылка 1 — ложь. Умозаключение формально верное, вывод — ложный.*

**16–22' | Физкультминутка-логика: «Охотник стоит»**

Учитель зачитывает умозаключение. Если форма правильная — встать. Неправильная — сидеть. Посылки не важны — только структура.

Примеры:
1. «Все птицы летают. Орёл — птица. → Орёл летает.» (правильная — встать; что скажете про пингвина? — потом обсудим)
2. «Все птицы летают. Самолёт летает. → Самолёт — птица.» (неправильная — сидеть)
3. «Некоторые спортсмены — чемпионы. Иван — спортсмен. → Иван — чемпион.» (неправильная — «некоторые» ≠ «все»)
4. «Ни одна рыба не является млекопитающим. Кит — млекопитающее. → Кит — не рыба.» (правильная — встать!)
5. «Все отличники учатся хорошо. Маша учится хорошо. → Маша — отличница.» (неправильная — сидеть)

**22–35' | Блок Б: пары строят умозаключения**
Формат: пары | Рабочий лист 1

Часть 1 (8 мин): из двух посылок написать вывод — или объяснить, почему вывод не следует.
- «Все реки текут. Волга течёт. → ?» (нет вывода — неправильная форма)
- «Ни один камень не дышит. Эта скала — камень. → ?» (Эта скала не дышит — правильно!)
- «Некоторые растения ядовиты. Ромашка — растение. → ?» (нельзя сделать вывод — «некоторые» не значит «все»)

Часть 2 (6 мин): самостоятельно придумать одно правильное умозаключение и одно с ошибкой в форме.

---

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**35–42' | Рефлексия и интрига**

Уровень «Новичок» открыт — отметить в карточке.

Рефлексия: *«Почему умозаключение Б ("кот — животное → кот — собака") кажется почти правдой? Что именно обманывает наш мозг?»*

Интрига: *«На следующем занятии — умозаключения, с которыми лучшие математики человечества боролись веками и не победили. Подготовьтесь.»*

### Занятие 2: «Парадоксы: когда мозг взрывается»

**Продолжительность:** 40–45 минут
**Цель:** познакомить с классическими парадоксами; понять, что такое самореференция и бесконечный регресс

---

#### Акт I: ЗАХВАТ (7 минут)

**0–3' | Парадокс лжеца — первый удар**

Учитель пишет на доске одно предложение:

> **«Это предложение ложно.»**

*«Секунду подумайте. Оно истинное или ложное?»*

Пауза 30 секунд. Ученики пытаются. Потом поднимают руки — и оказывается, что ответа нет.

*«Если оно истинное — то оно ложное. Если оно ложное — то оно истинное. Это — парадокс лжеца. Ему 2500 лет. Его до сих пор никто не решил.»*

**3–7' | Первая победа: история Эпименида**

Учитель рассказывает:

> *Эпименид был критянином и сказал: «Все критяне — лжецы». Если он говорит правду — значит все критяне, включая его, лжецы — значит он лжёт. Если он лжёт — значит не все критяне лжецы — значит он может говорить правду...*

Вопрос классу: *«Это та же ловушка? Чем она похожа?»* Ключевое слово — **самореференция**: суждение говорит о самом себе.

---

#### Акт II: РАЗВИТИЕ (28 минут)

**7–17' | Блок А: три парадокса античности**
Формат: фронтально | Техника: живые сценки + обсуждение

**Парадокс 1 — Ахиллес и черепаха (Зенон, V в. до н.э.)**

Разыгрывается в классе: два ученика — Ахиллес и Черепаха. Черепаха стартует с форы 10 шагов. Ахиллес бежит в 10 раз быстрее.

Учитель ведёт логику:
> «Ахиллес пробежал 10 шагов — черепаха прошла 1. Ахиллес пробежал ещё 1 шаг — черепаха прошла 0,1. Ещё 0,1 шага — черепаха прошла 0,01... Ахиллес *никогда* не догонит черепаху — он бесконечно приближается, но никогда не достигает!»

Класс: но это же абсурд, мы видим, что в реальности Ахиллес обгоняет!

*«Именно. Рассуждение верное по форме — но противоречит реальности. Где ловушка?»* (бесконечное деление пространства ≠ бесконечное время; греки не знали о сходимости рядов). Ключевое слово: **бесконечный регресс**.

**Парадокс 2 — Куча (Сорит, IV в. до н.э.)**

Учитель кладёт на стол книги (или рисует кружки):

> «1000 зёрен — куча? (Да.) Убираем одно → 999. Куча? (Да.) Убираем ещё... 500? 100? 10? 5? 2? 1?»

Ученики начинают спорить: где граница? Когда именно куча перестаёт быть кучей?

Ответ: **нет чёткой границы**. Это парадокс расплывчатых понятий. Применения к жизни:
- С какого возраста человек «старый»?
- Сколько волос нужно потерять, чтобы стать «лысым»?
- Когда человек становится «высоким»?

Ключевое слово: **нечёткие понятия** (размытые границы категорий).

**17–22' | Физкультминутка-логика: «Граница»**

Учитель называет понятие и число/степень. Ученики голосуют ногами: шаг влево = «да, попадает», шаг вправо = «нет, не попадает».

- «Человек ростом 185 см — высокий?» → большинство шагает влево
- «Человек ростом 170 см — высокий?» → раскол в классе
- «Человек ростом 165 см — высокий?» → раскол ещё сильнее

*«Вот где живёт парадокс кучи — прямо в нашем языке.»*

**22–35' | Блок Б: исследование парадоксов в парах**
Формат: пары | Рабочий лист 2

Каждой паре — один из трёх мини-парадоксов. Задача: объяснить «ловушку» своими словами.

1. «Если бог всемогущ — может ли он создать камень, который сам не сможет поднять?»
2. «В деревне есть парикмахер, который бреет всех, кто не бреется сам — и только их. Кто бреет парикмахера?»
3. «Одна стрела летит. В каждый момент времени стрела занимает чёткое положение — то есть она неподвижна. Но из бесконечного количества "неподвижных" моментов складывается... движение?»

После — пары объясняют классу. Учитель добавляет: что из этого решено наукой, что нет.

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**35–42' | Уровень «Следопыт» + рефлексия**

Уровень открыт — отметить в карточке.

Рефлексия — момент несогласия: *«Парадоксы — это серьёзная проблема или просто игра слов? Кто считает, что это просто языковая путаница, — поднимите руку. Кто думает, что за ними стоит настоящая математическая проблема?»*

Интрига: *«Некоторые люди поняли, что парадоксы — это оружие. И начали им пользоваться. Следующее занятие — их история.»*

### Занятие 3: «Рождение обмана: как греки придумали логический обман»

**Продолжительность:** 40–45 минут
**Цель:** познакомить с историей и механизмами классических софизмов; научить отличать намеренную ошибку от искренней

---

#### Акт I: ЗАХВАТ (7 минут)

**0–3' | Первый купец — первый трюк**

Учитель рассказывает историю:

> *Древняя Греция, рынок. Протагор — первый платный учитель риторики. К нему приходит бедный студент Эватл: «Хочу учиться спорить. Но денег нет.» Протагор предлагает: «Учись бесплатно. Заплатишь после первого выигранного суда.»*

*Эватл учится. Заканчивает. И... никогда не берётся за дела в суде. Протагор подаёт на него в суд.*

Учитель: *«Протагор говорит: "Если ты выиграешь суд — ты заплатишь по договору (ты выиграл первое дело). Если проиграешь — заплатишь по решению суда." Эватл отвечает: "Если я выиграю — я не платил по решению суда. Если проиграю — я не выиграл первого дела, значит не плачу по договору."»*

*Кто прав?*

**3–7' | Первая победа: именуем ловушку**

Разбираем: оба рассуждения формально правильные. Ловушка — в двусмысленности слова «выиграть» (в суде как решение vs. в суде как первый опыт). Это называется **амфиболия** — игра на двусмысленности.

---

#### Акт II: РАЗВИТИЕ (28 минут)

**7–17' | Блок А: пять классических приёмов софистов**
Формат: фронтально | Техника: примеры + именование

Учитель разбирает пять приёмов. После каждого — ученики сами придумывают пример.

**1. Амфиболия** — двусмысленность термина

> «Я обещал отдать долг, когда смогу. Я пока не могу.» (слово «смогу» — финансово или физически?)

**2. Подмена тезиса** — начали доказывать одно, закончили другим

> «Ты говоришь, что домашние задания полезны. Но ты сам иногда делаешь ошибки. Значит, ты не прав.» (тезис был про задания — а атакован человек)

**3. Порочный круг** — вывод уже содержится в посылке

> «Эта книга правдива, потому что в ней написана правда.» (а почему в ней правда? — потому что книга правдива)

**4. Поспешное обобщение** — из частного сделан всеобщий вывод

> «Я дважды был обманут на этом рынке. Все торговцы — мошенники.»

**5. Ложная причина** (пост хок)

> «После того как избрали нового мэра, цены выросли. Значит, он виноват в росте цен.» (после ≠ вследствие)

**17–22' | Физкультминутка-логика: «Назови приём»**

Учитель зачитывает короткое рассуждение. Ученики встают и называют приём хором — или молчат, если ни один не подходит (честная ошибка).

5 примеров — темп быстрый, соревновательный.

**22–35' | Блок Б: «Мошенник или ошибка?» — работа в группах**
Формат: группы по 3–4 человека | Рабочий лист 3

Группы получают 6 рассуждений. Задача — заполнить таблицу:

| Рассуждение | Ошибка есть? | Тип ошибки | Намеренный обман или нет? | Как исправить? |
|-------------|-------------|-----------|--------------------------|----------------|
| ... | | | | |

Ключевая дискуссия: как отличить намеренный софизм от добросовестной ошибки? Критерии:
- Знает ли человек о неверности?
- Выгодно ли ему это рассуждение?
- Уходит ли он от вопроса или отвечает?

---

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**35–42' | Уровень «Мастер» + рефлексия**

Уровень открыт — отметить в карточке.

Рефлексия — момент несогласия: *«Кто считает, что знание этих приёмов делает человека лучше — а кто считает, что это опасное знание? Почему?»*

---

### Занятие 4: «Охота в дикой природе: софизмы сегодня»

**Продолжительность:** 40–45 минут
**Цель:** научиться распознавать живые примеры из рекламы, политики, соцсетей

---

#### Акт I: ЗАХВАТ (7 минут)

**0–2' | Реальный трофей**

Учитель показывает (или зачитывает) реальную рекламу / заголовок новости / пост из соцсетей — заранее подобранный.

*«Это — живой экземпляр. Через 45 минут вы его разоберёте по косточкам. А пока угадайте: какой приём здесь использован?»*

**2–7' | Первая победа: быстрая охота**

Пять «быстрых» примеров — ученики угадывают приём без подсказки (проверка знаний с прошлого урока).

---

#### Акт II: РАЗВИТИЕ (28 минут)

**7–17' | Блок А: новые охотничьи угодья**
Формат: фронтально | Техника: категории + живые примеры

**Реклама:**

- *«9 из 10 стоматологов рекомендуют эту зубную пасту»* → каких десяти? когда? что именно рекомендуют? (манипуляция статистикой)
- *«Натуральный состав — значит безопасный»* → цианид тоже натуральный (апелляция к природному)
- *«Все знают, что это лучший бренд»* → "все знают" — кто эти все? (апелляция к большинству)

**Соцсети и мессенджеры:**

- *«Поделись этим постом, иначе ты не уважаешь ветеранов / больных детей / родину»* → шантаж чувством вины без логики (апелляция к эмоциям)
- *«Этот блогер говорит неправду — посмотрите, какая у него машина»* → атака на личность вместо аргументов (ad hominem)
- *«Ты слишком молод, чтобы иметь своё мнение»* → дискредитация по возрасту (апелляция к авторитету наоборот)

**Новости:**

- *«Уровень преступности вырос на 50%»* → с 2 случаев до 3? Или с 1000 до 1500? Абсолютное vs. относительное (манипуляция цифрами)
- *«Эксперты считают...»* → какие эксперты? сколько? в какой области? (апелляция к анонимному авторитету)

**Повседневная жизнь:**

- *«Если разрешить велосипеды в парке — скоро там будут мотоциклы, потом машины, потом гонки»* → скользкий склон
- *«Ты сам поступал также — зачем меня критикуешь?»* → tu quoque (и ты тоже), не отвечает на аргумент

**17–22' | Физкультминутка-логика: «Живая охота»**

Ученики встают. Учитель называет приём — ученики за 10 секунд сами придумывают пример из жизни и называют соседу. Потом 2–3 самых ярких — вслух классу.

**22–35' | Блок Б: полевое исследование — индивидуальная охота**
Формат: индивидуально + обсуждение | Рабочий лист 4

Задание А: разобрать «реальный трофей» из захвата по «5 шагам охотника» — письменно.

Задание Б: найти или вспомнить собственный пример из жизни — рекламу, фразу из разговора, пост. Записать: какой приём, почему это ловушка, как возразить.

Задание В (звёздочка): составить самостоятельно один убедительно-звучащий софизм — и объяснить, как его разоблачить.

---

#### Акт III: КУЛЬМИНАЦИЯ (5 минут)

**35–42' | Уровень «Эксперт» + peer review**

Ученики обмениваются рабочими листами — оценивают разбор соседа: правильно ли назван приём?

Рефлексия: *«Если вы теперь знаете эти приёмы — вы защищены от них? Или, наоборот, можете ими пользоваться? Что с этим делать?»*

---

### Занятие 5: «Финальный турнир Клуба»

**Продолжительность:** 40–45 минут
**Цель:** применить «5 шагов охотника» в командном формате; итоговая рефлексия

---

#### Акт I: ЗАХВАТ (5 минут)

**0–5'** Торжественный «вход в Клуб». Карточки уровней — на столах. Учитель объявляет регламент турнира.

> *«Три раунда. В каждом — одна команда защищает рассуждение, другая — атакует. Побеждает тот, кто точнее называет ловушку и убедительнее объясняет. Жюри — остальные команды. Критерии: точность (назван приём?), аргументация (почему это ошибка?), ясность (все поняли объяснение?).»*

---

#### Акт II: ТУРНИР (30 минут)

**Раунд 1 — Классика (8 мин)**

Карточка: известное историческое рассуждение (один из силлогизмов Аристотеля, намеренно искажённый). Команда А «защищает» (объясняет, почему оно кажется правильным). Команда Б «атакует» (находит ошибку по 5 шагам).

**Раунд 2 — Современность (8 мин)**

Карточка: реальный текст — фрагмент рекламы, заголовок новости, цитата из соцсети (не оскорбительная, не политически острая). Задача та же, роли меняются.

**Раунд 3 — Самостоятельный (12 мин)**

Команды получают 3 минуты — создают собственное рассуждение с намеренной ошибкой. Соперники пытаются разоблачить. Побеждает тот, чья ловушка оказалась самой незаметной — но была корректно разоблачена при разборе.

---

#### Акт III: КУЛЬМИНАЦИЯ (8 минут)

**Уровень «Легенда» + ачивка «Член Клуба»**

Все ученики, прошедшие турнир, — члены Клуба. Вручение ачивок.

**Финальная рефлексия курса «5 шагов»:**

Каждый ученик заполняет «Охотничью карточку»:

```
╔═══════════════════════════════════════════════╗
║        МОЯ ОХОТНИЧЬЯ КАРТОЧКА                 ║
╠═══════════════════════════════════════════════╣
║ Самая удивительная ловушка, которую я узнал:  ║
║ _____________________________________________ ║
║                                               ║
║ Пример из моей жизни, который я теперь вижу   ║
║ иначе:                                        ║
║ _____________________________________________ ║
║                                               ║
║ Самый трудный вопрос, который у меня остался: ║
║ _____________________________________________ ║
╚═══════════════════════════════════════════════╝
```

## 5. Контрольная точка

### 5.1. Задания

**Задание А (базовый уровень):**

Прочитай рассуждение и ответь на вопросы (5 шагов охотника):

> «Спортсмены — здоровые люди. Иван занимается спортом. Значит, Иван здоровый человек.»

1. Запиши посылки и вывод.
2. Правильна ли форма умозаключения?
3. Правдивы ли посылки?
4. Следует ли вывод?

> «Все успешные предприниматели рано встают. Я рано встаю. Значит, я стану успешным предпринимателем.»

5. Найди ошибку в форме — объясни своими словами.

**Задание Б (продвинутый уровень):**

Найди и назови логический приём в каждом из трёх текстов:

1. *«Доктора рекомендуют пить воду — значит, чем больше воды, тем лучше»*
2. *«Ты говоришь, что надо экономить, но сам купил новый телефон»*
3. *«Если мы снизим оценки за поведение — скоро в школах будет анархия»*

**Задание ★ (олимпиадный уровень):**

Объясни парадокс брадобрея (Рассела) — и напиши, почему его нельзя решить в рамках обычной двузначной логики (истинно / ложно).

### 5.2. Уровни достижений

| Уровень | Условие | Ачивка |
|---------|---------|--------|
| Базовый | Задание А: структура умозаключения верно разобрана | «Член Клуба» |
| Продвинутый | А + Б: три приёма названы с объяснением | «Член Клуба» + «Следопыт класса» |
| Олимпиадный | А + Б + ★ | «Эксперт Клуба» |

### 5.3. Если ученик не справился

- Проработать занятие 1 через конкретные примеры из жизни ученика
- Упрощённое задание А': только одно умозаключение — разобрать по шагам 1–3 (без шага 4 и 5)
- Парное разоблачение: ученик работает в паре с «охотником» постарше / сильнее

## 6. Банк заданий

### 6.1. Базовый уровень

**Б-01.** Запишите умозаключение в форме «П1 + П2 → В» для: «Все планеты вращаются вокруг звезды. Земля — планета.»

**Б-02.** Правильна ли форма? «Некоторые кошки — рыжие. Васька — рыжий. → Васька — кошка.»

**Б-03.** Объясните парадокс лжеца в 2–3 предложениях.

**Б-04.** В чём ошибка: «После дождя вышло солнце. Значит, дождь вызвал солнце»?

**Б-05.** Назовите приём: «Этот учёный предлагает налоговую реформу, но он сам богатый — зачем слушать?»

**Б-06.** Почему «9 из 10 стоматологов рекомендуют» — это неполная информация?

**Б-07.** Как отличить мнение от суждения? Приведите пример каждого.

**Б-08.** Объясните, почему Ахиллес в задаче Зенона не может догнать черепаху — и почему в реальности может.

### 6.2. Продвинутый уровень

**П-01.** Составьте умозаключение с ложными посылками, но правильной формой. Объясните, почему оно «работает» логически, но даёт ложный вывод.

**П-02.** Разберите по «5 шагам охотника» рассуждение: «Все, кто ест мясо, болеют раком. Мой дед ел мясо и заболел раком. Значит, мясо вызвало его рак.»

**П-03.** Найдите два разных логических приёма в одном тексте: «Этот политик предлагает сократить налоги. Но он сам не платит налоги честно! А вообще, если снизим налоги — бюджет рухнет, пенсии исчезнут, страна развалится.»

**П-04.** Придумайте рекламный слоган, который использует апелляцию к природному — и объясните, как его разоблачить.

**П-05.** Парадокс кучи: предложите чёткое определение слова «высокий» (применительно к человеку), которое не создаёт парадокса. Возможно ли это?

### 6.3. Олимпиадный уровень

**О-01.** «Парадокс Рассела»: Существует множество всех множеств, которые не содержат себя как элемент. Входит ли это множество в себя? Объясните, почему это та же структура, что и парадокс брадобрея.

**О-02.** Античный парадокс: «Зерно падает беззвучно. Значит, мешок зерна тоже падает беззвучно.» Как называется эта ошибка? В чём её связь с парадоксом кучи?

**О-03.** Придумайте ситуацию из современной жизни, где один и тот же текст содержит одновременно: (а) поспешное обобщение, (б) апелляцию к авторитету и (в) скользкий склон. Объясните каждую ловушку.

**О-04.** Существует ли принципиальная разница между парадоксом и софизмом — или это одно и то же? Напишите развёрнутый ответ (8–10 предложений) с примерами.

## 7. Частые проблемы и решения

| Проблема | Причина | Решение |
|----------|---------|---------|
| Ученик путает ошибку в форме и ошибку в посылке | Не различает структуру и содержание | «Форма — это чертёж дома. Посылка — материал. Дом может быть построен по верному чертежу из плохого кирпича» |
| Считает, что все рассуждения, ведущие к верному выводу, правильны | Смотрит только на результат | Пример: «Кот — животное. Собака — животное. → Кот — собака» — обе посылки верны, вывод ложный |
| Не может отличить парадокс от противоречия | Нет понятия самореференции | «Парадокс — это когда сама структура суждения ломает логику. Попробуй применить к нему 5 шагов — они не работают» |
| Считает, что знание софизмов позволяет "правильно обманывать" | Неверная этическая установка | Обсудить: почему манипуляция разрушает доверие в долгосрочной перспективе. Пример из истории: Горгий прославился, но ему не доверяли |
| Трудно найти пример из жизни | Мало медиаопыта | Помочь подобрать: реклама в магазине, фразы из разговоров с родителями / друзьями, заголовки в интернете |
| На турнире атакует человека, а не аргумент | Эмоциональная реакция | «Атаковать аргумент — значит показать ошибку в логике. Атаковать человека — это и есть ad hominem, один из наших приёмов» |
| Парадокс Зенона кажется скучным | Непонятна связь с жизнью | Перевести: «Можно ли добраться до стены, если каждый шаг — половина оставшегося расстояния?» — попробовать физически |

## 8. Соответствие методологии «Живой урок 360»

| Принцип | Реализация |
|---------|------------|
| **Трёхактная структура** | Захват → Развитие → Кульминация в каждом занятии |
| **Ключевой вопрос** | «Как из правильных слов получить ложный вывод?» — возвращается в рефлексии каждого занятия |
| **Смена деятельности 7–10 мин** | Лекция ↔ физкультминутка ↔ пары / группы ↔ индивидуально |
| **Ученик > учитель (65%+)** | Блоки Б — самостоятельная работа; турнир — полностью ученики |
| **Физкультминутки-логика** | «Охотник стоит», «Граница», «Назови приём», «Живая охота» |
| **Реальный контекст** | Реклама, соцсети, новости, переписки — живые примеры на каждом уроке |
| **Момент несогласия** | Зан. 2: «парадокс — игра слов или реальная проблема?»; зан. 3: «опасное ли это знание?» |
| **Формулировка вслух** | Команды объясняют ловушки в турнире; пары защищают позицию |
| **Peer review** | Зан. 4: обмен рабочими листами; турнир: жюри из учеников |
| **Рефлексия** | Карточка уровня + «охотничья карточка» в финале |
| **Именной алгоритм** | «5 шагов охотника» — применяется в каждом занятии, от простых к сложным |
| **Геймификация (Пул Б)** | Уровни охотника (5 уровней, по занятию); ачивки «Член Клуба» / «Эксперт» |
| **Связи с модулями** | Л-СШ-03 (аргументы): шаги 1–2; Л-СШ-04 (ловушки): узнают «старых знакомых» в новом свете; Л-СШ-06 (дедукция): формальная логика умозаключений |
